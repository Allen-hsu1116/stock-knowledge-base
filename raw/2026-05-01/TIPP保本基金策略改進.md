# 保本基金策略：CPPI、TIPP、OBPI 三種策略比較

**來源**：https://m.jiemian.com/article/1130185_yidian.html
**日期**：2026-05-01
**主題**：風險管理

---

## 保本基金三種策略

1. **CPPI**：固定比例投資組合保險策略
2. **TIPP**：時間不變性投資組合保險策略
3. **OBPI**：基於期權的投資組合保險策略

## CPPI 詳解

最早由 Black、Jones 和 Perold 於1987年提出。基本原理是將固收類資產的預期收益作為安全墊投資於權益類資產，從而避免投資組合的本金損失。

設一個風險乘數（放大倍數），按固收類資產預期收益的一定比例確定權益資產的投資額度，從而在控制下方風險的同時，充分捕捉市場上漲帶來的投資收益。

**CPPI三步**：
1. 根據到期保本額和合理折現率確定投資組合的價值底線
2. 算出投資組合現時淨值超過價值底線的數額，即安全墊
3. 將安全墊乘以放大倍數投資於權益資產，剩餘投資於固收類資產

**舉例**：初始資金100萬，100%保本，無風險利率3%，風險乘數2，保本期3年。

- T=0：價值底線 = 100/(1.03)^3 = 91.52萬，安全墊 = 100-91.52 = 8.48萬，權益投資 = 8.48×2 = 16.96萬，固收投資 = 83.04萬
- T=1（權益+20%）：總資產 = 83.04×1.03 + 16.96×1.2 = 85.51+20.35 = 105.86萬，新安全墊 = 105.86-94.26 = 11.60萬，新權益投資 = 11.60×2 = 23.20萬
- T=2（權益-25%）：總資產 = 85.51×1.03 + 23.20×0.75 = 88.11+17.40 = 105.51萬，新安全墊 = 105.51-97.09 = 8.42萬，新權益投資 = 8.42×2 = 16.84萬

**CPPI特點**：「追漲殺跌」——股價上行時買入股票，股價下行時賣出股票，一定程度上放大市場波動性。

**關鍵因素**：
- 保本額度影響安全墊高低：保本額度越低→安全墊越高→權益可投資額度越大
- 風險乘數可由基金經理根據市場情形調整
- CPPI中保本額度是固定值，不隨投資組合價值增減變動

## TIPP 詳解

TIPP是對CPPI的改進，將保本額度與投資組合價值變動掛鉤，動態向上調整。

**核心差異**：當期的價值底線不再由保本額度貼現值決定，而是取當期保本額度和上期保本額度的較大值。

- 投資組合價值增加時→保本額度要求也提高→鎖定收益、落袋為安
- 投資風格更為穩健
- 避免前期投資收益在後續市場變動中損失殆盡

## OBPI 詳解

基於期權的投資組合保險策略，基本原理與CPPI相同，但將標的物中的權益資產替換為股指的看漲期權。大部分資金投資於國債等固收類資產使其到期價值等於保本金額，剩餘資金投資於股指看漲期權。

## 保本基金三大風險

1. **市場短期巨幅下跌**：流動性極度喪失，放大風險敞口來不及變現就擊穿保本線（如1987年黑色星期一，道瓊下跌22.6%）
2. **頻繁劇烈波動**：引發大量贖回，頻繁調整增加運作成本
3. **管理人風險**：未嚴格執行策略

## 新規限制（避險策略基金）

- 穩健資產配置比例 ≥ 80%
- 債券部分信用等級 AAA 以上
- 權益類資產 ≤ 安全墊的3倍
- 基金單位累計淨值低於保本金額超過2%須報備
- 每3個月壓力測試
