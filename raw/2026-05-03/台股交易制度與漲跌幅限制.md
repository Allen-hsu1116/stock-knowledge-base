## 證交所官方交易制度

### 一般交易

- 交易時間：週一至週五，撮合成交時間9:00至13:30
- 委託時間：8:30至13:30
- 開盤前1分鐘暫緩開盤機制（8:59-9:00模擬撮合價漲跌超3.5%）
- 收盤前1分鐘暫緩收盤機制（13:29-13:30同理）

### 交易單位

- 股票：1,000股（1張）
- ETF：1,000單位
- 外國股票第二上市：不以1,000股為限
- 轉換公司債：面額100,000元

### 漲跌幅限制

- 股票、ETF、ETN：±10%（以當日開盤競價基準計算）
- 槓桿反向ETF：10%×倍數（2倍型±20%，-1倍型±20%）
- 權證：依標的證券漲跌幅×行使比例計算
- 公司債：±5%
- 新上市普通股首5日：無漲跌幅限制
- 外國股票第二上市、追蹤國外指數ETF：無漲跌幅限制

### 升降單位（跳動單位）

- 股價<5元：0.01元
- 5-10元：0.05元
- 10-50元：0.05元
- 50-100元：0.10元
- 100-150元：0.50元
- 150-500元：1.00元
- 500-1000元：1.00元
- >1000元：5.00元

### 委託方式（六種）

1. 限價ROD（當日有效）
2. 限價IOC（立即成交或取消）
3. 限價FOK（全部成交或取消）
4. 市價ROD（當日有效）
5. 市價IOC（立即成交或取消）
6. 市價FOK（全部成交或取消）

### 競價方式

- 開盤（8:30-9:00）：集合競價，僅限限價ROD
- 盤中（9:00-13:25）：逐筆交易，六種委託皆可
- 收盤（13:25-13:30）：集合競價，僅限限價ROD

### 盤中瞬間價格穩定措施

- 9:00-9:05：開盤價±3.5%
- 9:05-13:25：前5分鐘加權平均價±3.5%
- 觸發時暫緩撮合2分鐘後以集合競價撮合
- 權證、新上市首5日、低價股（基準<1元）不適用

### 零股交易

- 盤中零股：9:00-13:30，每5秒撮合一次（2024/12/2起）
- 盤後零股：13:40-14:30送委託，14:30集合競價一次撮合
- 升降單位、漲跌幅比照普通交易
- 零股成交價不作為當日開盤、收盤價格紀錄

### 盤後定價交易

- 收盤後以收盤價進行交易
- 集合競價方式一次撮合

### 新上市首5日

- 無漲跌幅限制
- 開盤收盤仍採集合競價
- 不實施盤中瞬間價格穩定措施

## 市場先生：漲停跌停實戰

### 漲停是什麼

- 當日漲幅達10%（以前一日收盤價計算），股價凍住不再上漲
- 例：台積電昨收600元，漲停價=600×1.1=660元
- 漲停原因：很多人想買，很少人想賣

### 跌停是什麼

- 當日跌幅達10%，股價凍住不再下跌
- 例：台積電昨收600元，跌停價=600×0.9=540元
- 跌停原因：很多人想賣，很少人想買

### 漲停鎖死 vs 跌停鎖死

- 漲停鎖死：沒有人掛賣單，大量買單掛在漲停價
- 跌停鎖死：沒有人掛買單，大量賣單掛在跌停價

### 漲跌停時的實戰策略

**手上有漲停股**：不建議賣，隔天續漲機率較高
**手上有跌停股**：賣不掉（流動性喪失），應看買進理由是否消失
**沒有漲停股想追**：不建議買，成本已貴10%，且可能買不到
**沒有跌停股想接**：看跌停原因，非個股長期因素+營運本質沒影響才考慮

### 漲跌停鎖死的替代方案

1. 盤後定價交易
2. 海外商品（ADR、相關期貨）
3. 衍生性商品（個股期貨、選擇權、權證）
4. 關聯性高的替代標的

### 各國漲跌幅/熔斷機制比較

| 國家 | 機制 |
|------|------|
| 美國 | S&P500跌7%/13%暫停15分鐘，跌20%停止交易 |
| 英國 | 富時100成分股漲跌超過動態參考價3%或靜態參考價8%，5分鐘集合競價 |
| 韓國 | KOSPI跌10%以上持續1分鐘暫停10分鐘，個股±15% |
| 日本 | 波動超過一定幅度停止10分鐘（依股價不同） |
| 香港 | 5分鐘內漲跌10%觸發5分鐘冷靜期 |
| 台灣 | 個股±10%，新上市首5日無限制 |
| 中國 | 個股±10% |
