# 如何用DCF(現金流量折現法)推估一家公司的真正價值？

**主題**：基本面分析

## 三種公司價值評估方式

### 1. 所有者權益（股東權益）法

一家公司合理的售價（價值）= 總資產-總負債= 所有者權益= 公司淨資產

每股價格=公司淨資產/公司所有的股份總數

問題：公司大部分的資產，都是以成本價登錄到資產負債表上的。萬一這家公司擁有非常值錢的土地、建築物或是特殊儀器等設備，用成本價賣出就虧大了。

### 2. 清算價值方式（以市價來估該公司所有的資產價值法）

將公司持有的土地、建築物、機械設備等資產，用市價方式類比賣出後的所得金額=市價估算的總資產

這家公司合理的價值=市價估算的總資產價值-總負債

問題：如果一家企業幾乎沒什麼有形資產，只有一群厲害的技術人員進行創業經營，卻能創造很多獲利，人與技術的價值無法在資產負債表上表現出來。

### 3. DCF法（Discounted Cash Flow) 現金流量折現法

「Discounted」的意思就是，把這家公司未來會產生的所有現金流，都換算成現在的價值。

因為「時間」就是金錢。今天的100萬和10年後的100萬，價值大不同。

DCF就是「打折之後的現金流量」——把未來的現金流量，用利率（折現率）進行打折，折算成今天價值(現值)的一種方法。

## DCF計算實例

假設一家公司未來五年現金流量分別為36w、54w、72w、112.5w與198w，用5%利率折現：

- 第一年 36w → 現值 34.29w
- 第二年 54w → 現值 48.98w
- 第三年 72w → 現值 62.20w
- 第四年 112.5w → 現值 92.55w
- 第五年 198w → 現值 155.14w

五年現金流淨現值合計=393.16w

這家公司如果要出價收購，最多只能出到393.16w，超過就是買貴了。

假設100股，每股價值=3.9316W/股

## 折現率的影響

同樣五年現金流，改變r(利率)：

- r = 5%，DCF = 393.16w
- r = 4%，DCF = 407.46w
- r = 3%，DCF = 422.49w
- r = 6%，DCF = 379.54w
- r = 7%，DCF = 366.58w
- r = 8%，DCF = 354.23w

利率和股價是反向關係：
- 央行降利率 → 股市利多（r下降，DCF上升）
- 央行升利率 → 股市利空（r上升，DCF下降）

## DCF三個核心要素

1. 現金流量的金額
2. 利率（折現率）— 最為重要，往往也最困難
3. 發生現金流的期間

## 常見誤區

1. 營收不是最重要的因素，帳上營收大不代表真有賺錢
2. 一家公司的價值來自現金流量（現金流量表是三大報表重中之重）
3. 一家公司的價值來自是否能創造源源不絕的利潤與現金流，不是商業模式的名稱
