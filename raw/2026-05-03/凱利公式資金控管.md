# 凱利公式-賭博與投機者的資金控管經典 - 勳仔的理財小角落

凱利公式在1956年由物理學家John Larry Kelly為了降低信息在傳輸過程中，因為噪音干擾引起信息傳輸錯誤的機率所提出的理論。發表在AT&T貝爾實驗室的貝爾技術期刊，最初是信息理論相關的學術文章，但因為公式著眼的問題剛好可以類比成賭局的場景，有人用它應用在21點的賽局理論，從而聲名大噪。

## 凱利公式 (Kelly Criterion)

公式：f = (p(b+1) - 1) / b

- p是勝率
- b是賠率（扣掉下注本金的部分）
- f是要下注的金額占總資金的百分比

舉例：獲勝機率60%(p=0.6)，賠率一賠一(b=1)
f = (0.6(1+1) - 1) / 1 = 0.2 = 20%

凱利公式假設資金可以無限不斷分割，所以永遠不可能破產。但實際貨幣有最小計量單位，如果遭遇連續虧損讓本金低到最小單位，公式就無法繼續執行。

## 凱利公式應用在期貨交易

當擁有一個交易策略後，可以先利用回測取得：
- 勝率（p）
- 盈虧比（b）

舉例：策略回測100次，45次賺錢(勝率45%)，平均獲利200元；55次虧錢，平均虧損100元，盈虧比2。

代入凱利公式：
F = (0.45(2+1) - 1) / 2 = 0.175 = 17.5%

也可以反向操作：把f設為0，固定勝率求最低盈虧比。
勝率45%時：0 = (0.45(b+1) - 1) / b，b = 1.22
代表勝率45%時，盈虧比至少要1.22才不會長期虧損。

## 凱利公式是交易聖杯嗎？

凱利公式在21點或骰子的賭局上，因為規則和機率是明確固定的，用數學推導出來的確是不會輸的策略。

但用在交易上有三大問題：
1. 交易策略的勝率來自過去回測，不同回測區間得到的參數不同
2. 回測區間的時空背景不同，勝率和盈虧比不是固定值
3. 如果策略穩定性差（參數標準差大），凱利公式求出的配置比例差異極大

凱利公式的核心概念：
- 贏錢才開始把部位放大
- 越是虧損，投入的部位就會變小
- 藉此保全在市場存活機率
- 長期才有獲勝的可能
