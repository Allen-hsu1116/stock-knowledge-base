## 價量分析的核心

對投資房地產的投資人來說，最重要的三件事就是 - 地點、地點、地點；而對投資股票的投資人，最重要的三件事是 - 時機、時機、時機。

價格與成交量之間邏輯關係的方法，是無法完全掌握交易資料背後的真相。成交量與價格相結合就會產生令人興奮的答案，價量分析就是市場交易行為分析的方法。

股市有句諺語說，老手會看亮，高手算籌碼，贏家諳趨勢。

## 主力大戶的操作邏輯

主力大戶日常工作就是讓所持的股票維持交易，並掌握市場上買賣雙方的籌碼流動狀態，以及供需力量之間的平衡。

### 出貨階段（賣出高峰期）
- 主力持股滿倉，市場籌碼夠乾淨時，開始在盤中左手小小賣出、右手小小買回來造市
- 價格緩緩墊高，量也逐漸放大
- 靠近計畫出貨價格區間邊緣時，左手小賣、右手大買來急拉股價引起市場注意
- 利用媒體不斷放出利多消息，製造不買會後悔的情緒
- 大批散戶搶進時，主力盤前試搓及早盤維持供不應求局面，盤中累積足夠數量後一次倒出
- 高檔爆大量 + 長上影線 = 主力大量倒貨
- 大黑K出現 = 主力貨要倒完了 → 形成左肩
- 後知後覺投資人倒給不知覺追買的投資人 → 右肩出現
- 股價像自由落體般的直直往下

### 進貨階段（買入高峰期）
- 觸發投資人對虧損的擔心價格區間
- 市場快速進入價格暴跌階段
- 投資人恐慌性拋售後，主力紛紛介入買進
- 低檔爆大量的原因
- 市場下跌速度遠遠超過上漲速度的原因

## 橫盤震盪階段的支撐壓力

股市價格走勢圖中，有七、八成的時間是處於橫盤震盪、區間整理的狀態，只有兩、三成的時間表現為上漲或下跌的趨勢。

瞭解橫盤震盪階段的支撐價位和壓力價位是價量分析的關鍵因素。

基於價格的支撐與壓力應配合成交量及時間整合分析：
- 當某檔股票保持在一定價格區間的時間越長，累積的成交量越大
- 只有當市場看跌或看漲情緒的平衡發生變化時，價格才會繼續前進
- 這個位置稱為「成交量的控制點」，是股票平衡的支點

### 成交量節點
- 低成交量節點：價格接近時，市場相對快速移動，因為之前累積的成交量很少
- 高成交量節點：價格變化可能暫停並進入震盪狀態

## 不同量價組合實例分析

### 價漲量增
- 智邦（2345）經過長期底部橫盤整理又出大量 = 波段行情要開始
- 帶量最好量是五盤均量的兩倍以上
- 這就是主力大戶正式啟動這檔股票的波段行情
- 高檔價漲又爆大量 = 主力大戶抬價出貨的信號，即將出現跌勢的前兆

### 價漲量縮/量平
- 上漲初期成交量不足，股價不易強勢上漲
- 如果收盤經常留著長長的上影線 = 有另一股勢力在製造賣壓
- 主力大戶拉抬意願不強 = 即將止漲的信號
- 高檔價漲量縮 + KD/RSI/MACD 背離 = 接下來的走勢多半要下跌

### 價跌量增
- 剛開始下跌初期，股價下跌又出量 = 股價仍會續跌
- 盤整已久才忽然出現大量（低檔爆大量換手）= 有主力大戶看中，已經進場吃貨
- V型反轉的低檔爆量 = 可列入考慮要買進的標的

### 價跌量縮
- 多頭行情時價跌量縮 = 很可能是主力在洗盤，做箱底測試
- 空頭行情時發生低成交量 = 有利於判斷股價是否落底
- 凹洞量：量縮到非常非常低，接著量又回復 → 發生在大盤指數時，代表大盤應該要反轉
- 窒息量：成交量不到20盤內最大成交量的10%以下 → 距離底部不遠了
- 不論凹洞量或窒息量，接著只要有足夠的量，股價就容易反轉向上

### 價平量平
- 股價下跌段價跌量平 = 股價應該會繼續下跌

### 量大不漲
- 成交量增加而股價卻不漲 = 通常都不是好現象
- 量大而不漲，股價準回頭
- 漲勢後期形成換手無力
- 價格不漲，量有但氣無力 = 行情陷入橫盤整理，小心盤久會跌
