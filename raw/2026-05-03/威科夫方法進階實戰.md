## 威科夫交易法核心四大法則

### 1. 市場週期四階段

**積累階段（Accumulation）**
- 價格在低位波動，成交量較低
- 狹窄的交易區間，形成底部區域
- 多重試探底部，但未能突破下方支撐
- 隨著主力逐步吸籌，成交量慢慢增加

**上升階段（Markup）**
- 價格突破積累階段的上方阻力位
- 成交量顯著放大，市場買盤強勁
- 回調幅度小，顯示市場強勢
- 高點和低點不斷抬高，形成上升通道

**分配階段（Distribution）**
- 價格在高位震盪，成交量放大
- 寬幅震盪，形成頂部區域
- 多重試探高點，但未能突破上方阻力
- 放量滯漲：儘管成交量較大，但價格上漲乏力

**下降階段（Markdown）**
- 價格跌破分配階段的下方支撐位
- 成交量顯著放大，賣盤壓力大
- 反彈幅度小，顯示市場疲軟
- 高點和低點不斷降低，形成下降通道

### 2. 供需法則

- 需求大於供給 → 價格上漲（量價齊升）
- 供給大於需求 → 價格下跌（量價齊跌）
- 供需平衡 → 價格橫盤（量增價穩）
- 價升量縮 → 上漲動力不足，趨勢可能結束
- 價跌量縮 → 下跌動力不足，可能反彈
- 量增價穩 → 主力可能在吸籌或派發

### 3. 因果法則

價格的運動幅度取決於之前的積累或分配的量。

- 長時間橫盤積累 → 突破後漲幅驚人（「橫有多長，豎有多高」）
- 短時間積累 → 突破後容易回落
- 假突破的根本原因：積累不足，持續上漲困難

### 4. 努力與結果法則

大的成交量應對應大的結果（大的價格波動）。

- 量價協調：大量對應大K線，趨勢健康
- 量價不協調：量大但價格變動小，預示趨勢轉變
- 高位大量成交但價格不漲 → 主力可能在出貨

## 市場結構

### 支撐和阻力

- 支撐位：主力吸籌的關鍵區域，價格接近時買盤增強
- 阻力位：主力派發的關鍵區域，價格接近時賣盤增強

### 交易範圍

- 積累範圍：價格在低位橫盤整理，主力悄然吸籌
- 分配範圍：價格在高位橫盤整理，主力逐步派發

### 假突破 vs 有效突破

- 假突破：短暫突破後迅速回落，主力製造假象誤導散戶
- 有效突破：突破後持續運行，成交量顯著放大

## 從主力視角看市場

### 主力操作四階段

1. **吸籌建倉**
   - 壓低建倉法：製造利空、大單賣出打壓，低價吸納
   - 橫盤建倉法：維持股價窄幅波動，隱蔽吸籌
   - 拉高建倉法：大幅買入推高股價，快速建倉

2. **洗盤**
   - 挖坑洗盤：製造下跌迫使不堅定投資者賣出
   - 橫盤洗盤：消磨耐心，清理浮籌
   - 高開低走洗盤：開盤拉高後迅速回落，套住短線

3. **拉升**
   - 震盪拉升：拉升和調整交替進行，穩步推高
   - 快速拉升：短期連續大單買入，迅速推高

4. **派發出貨**
   - 高位震盪派發：反覆小幅拉升和回調，逐步賣出
   - 快速派發：短時間大單賣出，迅速完成派發
   - 定向派發：大宗交易或協議轉讓給特定對象

### 散戶策略建議

- 不要模仿大莊家的左側抄底思維
- 選擇順勢右側交易，借莊家的勢
- 威科夫方法至少需要週線級別的時間維度
- 培養耐心和紀律性，制定合理的交易計劃

## 威科夫方法的缺點

- 需要較強的分析能力，每個人對因果關係的理解各不相同
- 主觀性強，同一套方法不同人使用結果差異大
- 至少需要週線級別時間維度，難以應用於短期交易
- 需要搭配其他技術指標和籌碼面分析確認
