## 均線是什麼？

均線就是過去一段時間的收盤價平均值，表示過去某段時間內的買賣雙方力道平衡點位，也可以說是這段時間的成本均價位置。

以5日均線為例：過去5天收盤價相加除以5。

### 均線種類（台灣股市常用）

- **5日均線（週線、5MA）**：極短線防守點，連續漲停股票跌破五日線就是極短線出場時機
- **10日均線（10MA）**：短線防守點，強勢股操作基準線，投信防守點是十日線
- **20日均線（月線、20MA）**：多頭股票防守點，跌破20日線進入短線空頭格局
- **60日均線（季線、60MA）**：中線防守點又稱生命線，月線跌破季線進入中期空頭修正
- **240日均線（年線、240MA）**：長線防守點，季線跌破年線進入長期空頭修正

## 均線扣抵是什麼？

均線扣抵所標記的位置為距今算起該均線的最舊一筆資料位置。例如5日均線的扣抵位置為距今算起5天前的地方。

### 均線扣抵使用方法

均線扣抵最厲害的地方是拿來做為預測未來均線趨勢的依據，進而推敲出未來股價的上漲與下跌機率。

**核心原理**：均線上揚為多頭走勢徵兆，均線下彎為空頭走勢徵兆。只要判斷出未來均線容易上揚或是下彎，就能推敲出未來股票價格的大致走勢。

**判斷方法**：
- 5日均線 = (A + B + C + D + E) / 5
- 隔天新5日均線 = (B + C + D + E + 新收盤價) / 5
- 被移除的是A（扣抵值），新增的是新收盤價
- 如果新收盤價 > A，均線上升
- 如果新收盤價 < A，均線下降
- 如果新收盤價 = A，均線持平

### 蹺蹺板原理快速記憶

當蹺蹺板左側扣抵值本來較高，逐漸往低處下降時，代表右側股價容易慢慢上升。反之蹺蹺板左側扣抵值本來較低，逐漸往高處上升，代表右側股價容易下降。

## 均線扣抵判斷支撐壓力力道

當股價由下準備來到均線附近時：
- **壓力力道弱**：該均線往後幾日要刪除的收盤價趨勢向下 → 只要下跌不太兇，均線自然會下彎，股價會自動突破
- **壓力力道強**：該均線往後幾日要刪除的收盤價趨勢向上 → 新收盤價上漲強度要強過扣抵值才可能突破

當股價由上向下測試支撐也是相同原理：
- **支撐力道弱**：扣抵值趨勢向上 → 均線自然上揚，支撐力道不足
- **支撐力道強**：扣抵值趨勢向下 → 均線自然下彎，支撐力道強勁

## 均線黃金交叉

當兩條不同週期的均線產生短週期均線由下向上穿越長週期均線，稱為「黃金交叉」，股價容易上漲；反之稱為「死亡交叉」，股價容易下跌。

**扣抵判斷黃金交叉**：
- 短期均線扣抵值呈現下降趨勢，長期均線扣抵值呈現上漲、盤整或下降較緩 → 未來出現黃金交叉機率高
- 短期均線扣抵值呈現上升、盤整或下降較緩於長期均線 → 黃金交叉機率低

## 實例：國際中橡（2104）2021/5/28

- 股價已站上5日和10日均線
- 20日均線扣抵值29.4元，股價24.9元，但扣抵值未來有半個月急跌 → 均線壓力疲弱，容易突破
- 60日均線扣抵值26.3元，但未來有2個月扣抵值盤整+上漲 → 均線壓力強勢，一次突破機率不高
- 預測：半個月內先突破20日均線 → 碰到60日均線壓力下跌 → 回到20日均線支撐 → 撐過2個月後突破60日均線

結果驗證：2021/6/11確實輕鬆突破20日均線，同天碰到60日均線被拉回，回到20日均線附近因扣抵值快速下降而出現支撐。

## 注意事項

1. 扣抵值方法不只適用均線，許多以週期計算的技術指標都能使用
2. 預測的都是「機率」，絕對會有不如預測的走勢
3. 當走勢與扣抵預測相反，代表不尋常事件發生，需留意原因
4. 長期持之以恆使用，邏輯和操作正確，方法不會讓你失望
