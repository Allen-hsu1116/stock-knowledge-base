## 什麼是 Chandelier Exit？

Chandelier Exit（吊燈出場）由 Charles Le Beau 開發，後來由 Alexander Elder 在《Come Into My Trading Room》一書中推廣。它是一種基於 ATR（平均真實波幅）的追蹤停損指標，用來判斷趨勢何時結束、何時該出場。

名稱由來：就像吊燈從天花板懸掛下來一樣，Chandelier Exit 從價格的最高點（做多）或最低點（做空）懸掛下來，距離由 ATR 決定。

## 計算公式

**做多（多頭部位）**：
Chandelier Exit (Long) = 最高價 - ATR(n) × 乘數

**做空（空頭部位）**：
Chandelier Exit (Short) = 最低價 + ATR(n) × 乘數

常用參數：
- ATR 期間：22（一個月交易日）
- 乘數：3（三倍 ATR 是 Le Beau 的原始設定）

## 核心邏輯

Chandelier Exit 結合了兩個關鍵要素：
1. **最高價/最低價**：追蹤趨勢的極值，確保停損點跟著趨勢走
2. **ATR**：用波動度動態調整距離，波動大時停損遠一點、波動小時停損近一點

這解決了固定百分比追蹤停損的最大問題：不同股票波動性不同，用同一個 % 數對高波動股太近（容易被洗）、對低波動股太遠（回吐太多）。

## 與其他追蹤停損方法比較

| 方法 | 基準點 | 距離計算 | 優點 | 缺點 |
|------|--------|----------|------|------|
| 固定%追蹤停損 | 最高價 | 固定百分比 | 簡單直覺 | 不考慮波動性 |
| SAR 拋物線 | 極值點 | 加速因子 | 自動加速 | 盤整假訊號多 |
| Chandelier Exit | 最高價 | ATR×乘數 | 動態適應波動 | 需調參數 |

## 實戰使用方式

### 做多出場
- 當收盤價跌破 Chandelier Exit 時，多頭部位出場
- Chandelier Exit 只會上移不會下移（做多時），確保鎖住利潤

### 做空出場
- 當收盤價漲破 Chandelier Exit 時，空頭部位出場
- Chandelier Exit 只會下移不會上移（做空時）

### 進場訊號（輔助）
雖然主要作為出場工具，但也有人用 Chandelier Exit 做進場判斷：
- 價格突破長期下降的 Chandelier Exit → 趨勢可能反轉做多
- 價格跌破長期上升的 Chandelier Exit → 趨勢可能反轉做空

## 參數調整建議

- **ATR 期間**：22（月線）是預設值，短線可用 10-14，長線可用 50
- **乘數**：3 是預設值，加大會讓停損更寬鬆（減少被洗但回吐更多），縮小會讓停損更緊（鎖利更快但更容易被洗出）
- 調整原則：先看標的最近 3 個月的 ATR 均值，確保乘數×ATR 大於常見回檔幅度

## 優缺點

### 優點
- 自動適應不同股票的波動性
- 不會被正常波動洗出場（ATR 動態調整）
- 明確的出場規則，減少情緒干擾
- 比固定%追蹤停損更合理

### 缺點
- 參數需要調整，沒有一體適用的設定
- 極端行情（如跳空漲停）ATR 會暴增，導致停損點突然變遠
- 盤整市中仍然會被來回洗（雖然比 SAR 好）
- 只提供出場訊號，不提供進場訊號

## 與 SAR 的關係

SAR 拋物線指標和 Chandelier Exit 都是追蹤停損工具，但機制不同：
- SAR 用加速因子（AF），一開始慢後來越來越快，適合趨勢明確的市場
- Chandelier Exit 用 ATR，速度穩定不會加速，適合波動較大的市場
- SAR 在盤整中假訊號更多（AF 會一直重置）
- Chandelier Exit 在盤整中表現較好（ATR 會自然縮小）
