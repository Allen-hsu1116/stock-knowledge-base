## 移動停利停損簡介

傳統停損停利的問題：
- 到達停利點後，股價仍持續走強，後續的獲利都沒有參與到
- 還沒到達停利點就開始下跌，最後碰到停損點只能忍痛出場

移動停利法主要就是能夠解決上述問題。移動停利（Trailing Stop），又稱移動鎖利、動態停利，主要是隨著獲利創新高後回檔特定 % 數或價格，來進行停利出場操作，是一種能夠避免大量獲利回吐下仍能守住獲利的出場策略。

移動停利的優勢在於能夠根據市場變化來自動調整，會隨著獲利創新高而逐步調整停利點的一種停利策略。

## 策略實作核心邏輯

1. 取得一段時間的價格高點（歷史 Candles API 取得策略開始後到最近一個交易日的最高價格）
2. 盤中即時價格若高於歷史最高價，最高價將會被即時價格取代
3. 當現價 <= 最高價 × (1 - 停利比例) 時，觸發平倉出場

### Python 實作關鍵程式碼

```python
# 移動停損停利判斷
def run_strategy(self, qty):
    if self.now_price is not None:
        # 盤中需判斷目前價位是否為更新價位
        if self.now_price >= self.high_price:
            self.high_price = self.now_price
        
        # 達到停利就進行平倉操作
        if self.now_price <= self.high_price * (1 - self.stopProfit_ratio):
            # 平倉
            self.sell_total_shares(qty)
```
