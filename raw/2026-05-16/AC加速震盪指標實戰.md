## AC指標概述

Accelerator Decelerator Oscillator (AC)，由Bill Williams發明，是Bill Williams五維交易系統的第三維指標，用於衡量市場驅動力的加速或減速。

Bill Williams的核心假設：**在價格改變方向之前，趨勢的動量會先減速**。換句話說，動量（速度）改變先於價格改變，加速度（動量的變化率）又先於動量改變。

AC是AO（Awesome Oscillator）的進階版本，用來偵測動量的加速度變化。

## 計算公式

```
中位價 = (最高價 + 最低價) / 2

AO = SMA(中位價, 5) - SMA(中位價, 34)

AC = AO - SMA(AO, 5)
```

AC = AO值 減去 AO的5期簡單移動平均

核心邏輯：
- AO衡量動量（速度）
- AC衡量動量的變化率（加速度）
- AC是AO的「一階導數」，AO是價格的「一階導數」

## AC指標判讀

### 零軸含義
- 零軸 = 市場驅動力與加速度的相對平衡點
- AC在零軸上方 = 上升動能正在加速
- AC在零軸下方 = 下降動能正在加速
- **零軸穿越本身不是交易訊號！**

### 柱狀圖顏色
- 綠色柱 = 動量正在增加（比前一根柱大）
- 紅色柱 = 動量正在減少（比前一根柱小）

### 四種狀態

1. **AC>0 且綠色柱上升**：多頭趨勢加速，買方更強，價格更容易上漲
2. **AC>0 且紅色柱下降**：多頭趨勢減速，可能轉向或進入盤整
3. **AC<0 且紅色柱下降**：空頭趨勢加速，賣方更強，價格更容易下跌
4. **AC<0 且綠色柱上升**：空頭趨勢減速，可能轉向或進入盤整

## 交易訊號

### 基本規則
- 當前柱為紅色 → 不買進，只能賣出或不交易
- 當前柱為綠色 → 不賣出，只能買進或不交易

### 順勢訊號（與加速度方向一致）— 較強

**買進訊號（AC在零軸上方）**：
- 零軸上方出現連續2根綠色柱（前可有1根紅色柱）
- 在訊號柱最高價上方1-3點設BuyStop

**賣出訊號（AC在零軸下方）**：
- 零軸下方出現連續2根紅色柱（前可有1根綠色柱）
- 在訊號柱最低價下方1-3點設SellStop

### 逆勢訊號（與加速度方向相反）— 較弱，需額外確認

**買進訊號（AC在零軸下方）**：
- 零軸下方需要連續3根綠色柱（前可有1根紅色柱）
- 若前2根綠色柱中有一根穿越零軸，則第3根即可買進
- 在訊號柱最高價上方1-3點設BuyStop

**賣出訊號（AC在零軸上方）**：
- 零軸上方需要連續3根紅色柱（前可有1根綠色柱）
- 若前2根紅色柱中有一根穿越零軸，則第3根即可賣出
- 在訊號柱最低價下方1-3點設SellStop

### 訊號取消規則
- 若在BuyStop/SellStop被觸發前，柱狀圖改變顏色 → 訊號作廢
- 若掛單距離轉折點夠遠且趨勢方向仍有效 → 可保留掛單

## 背離訊號

AC的背離訊號出現頻率比其他震盪指標低，但可靠性相對較高：
- 價格創新高但AC未創新高 → 上升動能減速，警惕反轉
- 價格創新低但AC未創新低 → 下降動能減速，警惕反轉

背離出現時，邏輯上應朝零軸突破方向操作，停損設在下一個局部極值後方。

## 參數調整建議

Williams原始參數(5, 34)是20多年前為期貨和股票設計的：
- 現代市場波動更大，建議根據標的和時間框架調整
- 活躍波動工具可嘗試(5, 20, 3)參數組合配合EMA
- 第一個參數（5）不建議更改，因與Elliott波浪結構和Mandelbrot碎形相關
- 低於4小時的時間框架假訊號太多，建議H1以上

## 盤整辨識

AC也可作為盤整市場的辨識工具：
- 低波動期間，AC柱狀圖在零軸附近小幅波動
- 綠紅交替頻繁且幅度極小 = 盤整狀態
- 此時不應使用AC的趨勢訊號

## 與其他Bill Williams指標的搭配

AC不建議單獨使用，應搭配其他Williams指標：
- **鱷魚線**：判斷趨勢方向，鱷魚睡覺時不用AC訊號
- **碎形**：碎形突破是主要訊號，AC是確認性質
- **AO動量震盪**：AO先發出訊號，AC確認加速度
- **MFI市場便利指標**：量價配合判斷

### AO + AC聯動邏輯
1. AO先轉向 → 動量開始改變
2. AC跟進同向 → 動量加速度確認
3. 兩者同向 = 高勝率進場點
4. AO轉向但AC未跟進 = 假訊號機率高
