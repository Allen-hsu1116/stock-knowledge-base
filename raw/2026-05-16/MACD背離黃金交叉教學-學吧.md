## 一、MACD 是什麼？

MACD，全名 Moving Average Convergence Divergence，中文叫「指數平滑異同移動平均線」。是一種「趨勢追蹤型指標」，本質上是均線的升級版，協助判斷市場趨勢的強度、方向、買賣時機。

核心概念：透過「快線與慢線的差距」來判斷趨勢，再用「柱狀體」顯示動能變化。

- 快線（DIF）：看短期趨勢，反應速度快
- 慢線（MACD 線，也叫 DEA）：中長期趨勢，反應速度慢
- 柱狀體（Histogram）：快線與慢線的差值，用來判斷多空力道

## 二、MACD 怎麼計算？

- 快線 DIF = 短期 EMA – 長期 EMA（常用12日和26日）
- 慢線 MACD = 取 DIF 的 9日 EMA
- 柱狀體 OSC = DIF – MACD 線

柱狀體顏色代表：
- 紅柱：DIF 在 MACD 線之上 → 動能偏多
- 綠柱：DIF 在 MACD 線之下 → 動能偏空

重要看點：
- 快線上穿慢線 → 多頭力道轉強
- 快線下破慢線 → 空頭力道增強
- 柱狀體變長 → 趨勢加速；縮小 → 動能減弱

## 三、MACD 怎麼看？

三個觀察重點：
1. 快慢線在零軸上/下 → 判斷大方向
2. 快線與慢線交叉 → 提供買賣訊號
3. 柱狀體顏色與長短 → 顯示多空力道

### 零軸判斷
- 快線、慢線都在零軸上方 → 多頭趨勢
- 快線、慢線都在零軸下方 → 空頭趨勢
- 零軸位置比交叉更重要，能幫助判斷大方向

### 黃金交叉與死亡交叉
- 黃金交叉：快線往上突破慢線，買進訊號。零軸之上出現最強。
- 死亡交叉：快線往下跌破慢線，賣出訊號。零軸之下出現最強。

### 柱狀體變化
- 紅柱變長 → 多頭趨勢加速
- 紅柱縮短 → 多頭動能減弱
- 綠柱變長 → 空頭趨勢加速
- 綠柱縮短 → 空頭動能減弱

## 四、MACD 背離：趨勢反轉的訊號

### 高檔背離（頂背離）
股價創新高，但 MACD 指標沒有同步創新高，甚至走弱。代表多頭力道衰退，常見於漲多後的高檔區。

### 低檔背離（底背離）
股價創新低，但 MACD 指標沒有再創新低，甚至反而抬頭。代表賣壓逐漸消退，常見於跌深後的低檔區。

### 背離的重點判斷
- 趨勢延續中出現背離 → 通常代表動能減弱，未必立即反轉
- 趨勢末端出現背離 → 反轉成功率最高
- 搭配成交量、K 線一起觀察 → 能大幅提升可靠度

## 五、MACD 的盲點與常見錯誤

1. **訊號落後**：MACD 由均線演化而來，屬於落後指標，看到黃金交叉時股價可能已經漲一段
2. **假交叉**：盤整行情中快線和慢線反覆交纏，出現一堆假訊號
3. **忽略大環境**：MACD 只能反映技術面，無法考慮消息面或基本面

## 六、MACD 的實戰搭配

1. **搭配均線**：股價在季線、半年線之上且均線多頭排列 → 黃金交叉更可靠
2. **搭配 K 線型態**：黃金交叉 + 紅K大陽線 → 趨勢延續；死亡交叉 + 黑K長上影線 → 空頭力道增強
3. **搭配成交量**：黃金交叉 + 放量 → 真突破機率高；黃金交叉 + 量縮 → 假突破機率高

## 七、新手用 MACD 的簡單原則

1. 順勢操作，不逆勢
2. 避免盤整行情
3. 先看柱狀體，再看交叉
4. 固定參數練習
