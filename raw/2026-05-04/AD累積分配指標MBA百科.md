## 什麼是累積/派發指標（A/D）

累積/派發指標（Accumulation/Distribution，簡稱A/D或AD）由Marc Chaikin開發，是一種量價指標，通過計算收盤價在當日價格區間中的位置來衡量資金流入流出情況。

AD指標的核心思想：收盤價越接近當日最高價，說明買方力量越強，累積（Accumulation）正在發生；收盤價越接近當日最低價，說明賣方力量越強，派發（Distribution）正在發生。

## 計算公式

步驟1：計算CLV（Close Location Value / 多空對比值）
CLV = ((收盤價 - 最低價) - (最高價 - 收盤價)) / (最高價 - 最低價)
簡化為：CLV = (2 × 收盤價 - 最高價 - 最低價) / (最高價 - 最低價)

CLV值介於 -1 到 +1 之間：
- 收盤價 = 最高價 → CLV = +1（最強多方）
- 收盤價 = 最低價 → CLV = -1（最強空方）
- 收盤價 = 中間 → CLV = 0

步驟2：計算資金流量乘數（MFV）
MFV = CLV × 當日成交量

步驟3：計算AD線
AD線 = 前日AD值 + 今日MFV

## AD 線判讀

1. **趨勢判斷**：AD線上升表示買方主導（累積），AD線下降表示賣方主導（派發）
2. **背離訊號**：股價創新高但AD線未創新高 → 頂背離，賣出訊號；股價創新低但AD線未創新低 → 底背離，買入訊號
3. **趨勢確認**：AD線與股價同方向移動，確認趨勢有效

## AD vs OBV 差異

| 特徵 | OBV | AD |
|------|-----|-----|
| 漲跌判定 | 只看收盤價漲跌 | 看收盤價在日內位置 |
| 漲時加量 | 全部成交量加入 | CLV加權後加入 |
| 跌時減量 | 全部成交量扣除 | CLV加權後扣除 |
| 平盤處理 | 不計算 | CLV可能不為0 |
| 資訊量 | 低（只有漲跌） | 高（有位置資訊） |

**關鍵差異**：OBV只看「漲還是跌」，AD看「漲多少/跌多少」。同樣是上漲日，收盤在最高價附近比收盤在半山腰的多方力量更強，OBV不區分這個，AD會區分。

## AD 的限制

1. **不考慮跳空缺口**：AD使用最高最低價計算CLV，但跳空缺口導致CLV無法正確反映真實的買賣力量
2. **高度依賴收盤價位置**：某些情況下收盤價不能完全代表當日買賣力量
3. **對冷門股效果差**：成交量過小時CLV的計算容易失真
