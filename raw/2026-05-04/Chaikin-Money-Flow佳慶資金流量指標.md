## 原文整理

### 什麼是 Chaikin Money Flow (CMF)

CMF 又稱「佳慶資金流量指標」、「蔡金貨幣流量」，由技術分析師 Marc Chaikin 根據「收集/派發線」(Accumulation/Distribution Line) 推陳出新的一種圖表分析指標。其原理與 MFI 很類似。

CMF 基於這樣的假設：強勢市場（上升趨勢）通常伴隨位於日最高價與最低價之間上半部分的收盤價以及放大的成交量。弱勢市場（下跌趨勢）通常伴隨位於日最高價與最低價之間下半部分的收盤價以及放大的成交量。

### CMF 計算公式

**步驟 1：計算 CLV（Close Location Value / 多空對比）**

CLV = ((收盤價 - 最低價) - (最高價 - 收盤價)) / (最高價 - 最低價)
簡化為：CLV = (2 × 收盤價 - 最高價 - 最低價) / (最高價 - 最低價)

當最高價 = 最低價時，CLV = 0

CLV 值介於 -1 到 +1 之間：
- 收盤價等於最高價時，CLV = +1
- 收盤價等於最低價時，CLV = -1
- 收盤價在最高最低價中間時，CLV = 0

**步驟 2：計算 CMF**

CMF = Σ(Vol × CLV) / Σ(Vol) （回溯期內加總）

預設回溯期為 20 日。

### CMF 判讀方法

1. **零軸分水嶺**：CMF > 0 市場處於牛市（買方主導），CMF < 0 市場處於熊市（賣方主導）
2. **停留時間**：CMF 在零軸上方（或下方）停留時間越長，趨勢越可靠
3. **±0.05 緩衝區**：直接穿零軉容易洗盤，建議用 +0.05 / -0.05 作為緩衝閾值，高於 +0.05 看多，低於 -0.05 看空
4. **超買超賣**：CMF 達 +0.3 視為超買，CMF 達 -0.3 視為超賣
5. **背離訊號**：CMF 與價格的背離具有重要意義，通常預示行情即將轉變
6. **長期收集訊號**：股票長期保持在零以上，表示有買家在長期收集。尤其是股價下跌時 CMF 回落到零就止跌回升，是牛市中的正常調整
7. **熊市短暫反彈**：CMF 每次升至零就調頭，顯示購買力弱不禁風，可能是熊市的短暫後抽

### CMF 核心概念

「價漲量增，價跌量縮」——當收盤價越靠近最高價，代表成交量發揮多方影響力，CMV 為正；反之當收盤價越靠近最低價，代表成交量受到空方影響力，CMV 為負。

### CMF vs MFI 差異

- CMF 使用 CLV（收盤價在日內位置）× 成交量
- MFI 使用典型價格（(H+L+C)/3）× 成交量，並加入漲跌判斷
- 兩者原理相似但計算方式不同
