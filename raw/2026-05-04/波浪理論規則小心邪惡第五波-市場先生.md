# 波浪理論規則是什麼？小心邪惡的波浪理論第五波

**主題**：技術分析

艾略特波浪理論是屬於技術分析的一種理論，發明者艾略特在研究歷史股價走勢變化，發現走勢形態上會不斷重複，上下上下的走勢很像海浪，因此命名為「波浪」。

波浪理論是一種用來預測未來股價趨勢的方式，它並非一種精確地計算，是一種比較框架性、架構性的概念和描述。

## 波浪理論的適用範圍

- 適用分析指數、外匯、期貨等大規模市場，容易被人為操控、股本小、個股不適用
- 可以了解在一段長時間中，目前市場是處在波浪中的哪一階段
- 主要用於中長期的走勢分析，不適用在過於短期的時間區間
- 以自然法則為主，不涉及人為干涉，在大型指數會更有意義

## 艾略特波浪理論的規則

完整一個波浪總共有 8 波，基本架構是由 5 個推動波（Impulse Waves）、3 個調整波（Corrective Waves）組成。

- 推動波：分為5個波浪，1、3、5是主要推動波，2、4是反向回檔修正波
- 調整波：分為3個波浪，A、C是主要調整波，B是反向修正波

## 波浪特性（群眾心理）

| 波浪 | 特性 |
|------|------|
| 波浪 1 | 通常不明顯，基本面消息幾乎是負面，市場情緒明顯看跌 |
| 波浪 2 | 波浪1的回檔，永遠不會超過波浪1的起點，成交量比波浪1小，價格通常不超過波浪1的61.8% |
| 波浪 3 | 道氏理論的買入信號，進入牛市，最大成交量與價格變動，5個推動波中最長 |
| 波浪 4 | 平緩盤整或回檔，底部不低於第1浪高點，可能長時間橫盤 |
| 波浪 5 | 波動度不定，市場情緒高漲，許多投資者買入在高點，被稱為「邪惡的第5波」 |
| 波浪 A | 投資人普遍認為只是短暫回檔，交易量在反向趨勢中增加 |
| 波浪 B | 假反彈，許多人進場搶低點 |
| 波浪 C | 市場意識到熊市，通常和波浪A一樣大，可延伸到A浪1.618倍 |

## 多頭行情基本規則

1. 波浪2的最低點不低於波浪1起點
2. 波浪3通常不會是最短的波段，且大多是主升段
3. 波浪1最高點與波浪4最低點不能重疊
4. 波浪b反彈上升高點，不會超過波浪a原本最高點

## 空頭行情基本規則

1. 波浪2的回檔一定會小於波浪1
2. 波浪3通常是推動波中幅度及時間最長、成交量最大
3. 波浪1最低點與波浪4最高點不能重疊
4. 波浪a最低點一定低於波浪b修正階段最低點

## 延伸波

- 出現在1、3、5波浪裡，其中1波，且只會出現1次
- 表示走勢非常強勁且趨勢較長
- 通常出現在第3波的機率高

## 邪惡的第5波

第5波可以存在很長的延伸波，市場已經大漲很長時間，新手紛紛進場，幅度甚至超越第3波。但隨後會迎來強烈反轉，前面漲多兇後面ABC波就殺多兇。

## 波浪計算方式（費波那契數列）

- 波浪2：波浪1的38.2%到78.6%之間
- 波浪3：1.618×或2.62×或4.25×波浪1
- 波浪4：波浪3的23.6%或38.2%或50%
- 波浪5：等於波浪1 或 1.618×波浪1 或 2.62×波浪1
- 波浪B：回檔A浪的38.2%到78.6%
- 波浪C：波浪A長度的1.382或1.618倍

市場先生提示：費波那契數列看看就好，波浪理論關鍵在於12345+ABC的結構框架，以及各階段對於背後群眾心理的解釋，至於波動走多少幅度、走多久時間，無法精準預估。

## 如何運用在交易中

- 使用波浪理論的最佳時間是猜測第3波剛形成，第2波結束後，第3波即將突破時進場
- 另一個時機點是第4波結束、第5波即將開始突破第3波前高時
- 波浪雖然事後看有那麼回事，事先預測其實很難精準
- 如果追蹤的任何一個波浪不符合規則，波浪就需要重新計算

## 注意事項

- 波浪模式並不一定會在市場上發生
- 建議與其他技術指標和工具相結合
- 1930年代提出的理論，市場心理學是否還合乎現今的投資心理，是可思考的觀點
- 波浪理論提供思考方式：從希望到興奮、從興奮到狂熱、從狂熱再到懷疑、從懷疑到恐慌、再從恐慌到絕望
- 不要試圖精準預測價格會走到哪裡，不要太把費波那契數列當成原則
