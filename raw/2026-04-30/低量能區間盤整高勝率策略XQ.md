## 原文

### 為什麼「低量+盤整」有助於提高勝率？

觀察：
- 股價較不存在溢價或膨脹的問題
- 持有成本較容易被控制在一定範圍內
- 近期個股的持有者對股價看法較一致
- 只要有人且有錢願意追價，趨勢就會特別容易形成

### 策略條件

轉換成3個條件：
- 成交量小於 800 張
- 成交量大於 5 張
- 區間未破底且創新高

進出場條件：獲利18%、停損18%

回測結果：勝率、最大回撤(MDD)和績效曲線等數據表現都稱得上非常亮眼。

核心精神：「人多的地方不要去」。

唯一美中不足的地方是，這個策略的當日交易胃納量極為受限（因為成交量極低）。補救方法：若選股結果位階在盤整區間附近，稍微等待一下拉回後買入，也還是很有機會的。

該策略對於資金部位不大的投資人來說，是一個不錯的策略雛形。

### XS 選股代碼

```
// 區間未破底且創新高
Input:period(60, "區間定義");
Input:percent(25, "區間振幅定義");

Condition1 = False;
Condition2 = False;

// 計算區間低點
Value1 = Lowest(Low, Period);

// 判斷區間低點不破
IF Value1 = Low[Period - 1]
Then Condition1 = True;

// 判斷區間高點 <= 區間振幅
IF Highest(High[1], Period) <= Value1 * (1 + Percent / 100)
Then Condition2 = True;

// 判斷區間低點不破 且 區間高點 <= 區間振幅
// 判斷收盤價 由下往上穿過 區間高點
IF Condition1 and Condition2
and Close Crosses Over Highest(Close[1], Period)
Then Ret=1;
```
