# 選擇權入門教學 – 選擇權、期權是什麼？該如何買賣？

選擇權(Options)又稱期權，是一種「未來可以用特定價格買賣商品的憑證」，選擇權買賣雙方會敲定契約、標的、履約價及買賣數量。

選擇權和買股票不大一樣，股票很單純就是買和賣而已，而選擇權還要加上時間(到期日)與約定價格(履約價)。

## 選擇權合約重點項目

- **標的**：選擇權契約對應標的，例如某股票或某指數
- **call/put**：買權call(到期可以用履約價買進)、賣權put(到期可以用履約價賣出)
- **履約價**：到期時契約的交易價格
- **到期日**：根據週期分成月選擇權或週選擇權
- **契約乘數**：不同標的1點對應的金額不同，例如小台指1點=50元

1個買賣交易單位稱為「一口」。選擇權的買賣因為是一份未來的合約，因此採用保證金交易。

## 選擇權實際交易例子

例如：買進1口10月份台指期10000點call月選擇權，目前市價55點

- 標的：加權指數
- call/put：買進call，認為到期時會漲遠超過10000點
- 履約價：10000點
- 到期日：10月份第3個星期三到期
- 契約乘數：1點50元
- 市價55點，買進需花費55點×50元/點=2750元權利金

如果到期結算時指數落在10500點，這個call的價值相當於500點，扣掉成本55點後獲利445點，相當於+22250元，獲利將近+800%。

但如果指數結算落在9950點，這個call一文不值，等於2750元虧損一空，即-100%。

## 選擇權4大特性

1. 槓桿高、風險大，容易大賺大賠
2. 可以做多，也可以做空
3. 買方虧損有限但獲利無上限，風險有限，但通常虧損機率大
4. 賣方虧損無限但獲利有上限，承擔所有風險，通常獲利機率大

## 什麼時候需要交易選擇權？

對行情的觀察不只是漲和跌，還有各種程度的看法：

| 程度 | 看法 |
|------|------|
| +3 | 大漲 |
| +2 | 小漲 |
| +1 | 難下跌 |
| 0 | 盤整 |
| -1 | 難上漲 |
| -2 | 小跌 |
| -3 | 大跌 |

選擇權是在任何情況之下，只要你能對未來預測得準，就能藉此獲利的工具。

案例1：未來一段時間不漲不跌機率很大
案例2：未來一個月絕不會漲超過10%
案例3：未來有90%機率上漲，但有10%機率超級大跌
案例4：未來可能超大漲幅或超大跌幅，但不會盤整

## 4種最基本的選擇權買賣方式

| 中文 | 英文 | 對未來的看法 |
|------|------|-------------|
| 買進買權 | Buy call | 認為未來會大漲 |
| 買進賣權 | Buy put | 認為未來會大跌 |
| 賣出買權 | Sell call | 認為未來不會大漲 |
| 賣出賣權 | Sell put | 認為未來不會大跌 |

## 買方vs賣方

買方沒有比較安全，賣方也沒有比較危險，端看你對未來的判斷是否精準。

賣方雖然風險無上限，但當賣方認為未來虧損的可能性越大，就會傾向收更貴的權利金才願意成交合約。

買方看似沒有風險，但一開始就要付出昂貴的權利金成本，因此最終也不見得會獲利。

## 選擇權風險

選擇權波動很大，動輒漲跌50%很正常。2018/10/11台指期一天下跌831點，盤中put月選擇權出現超過3000%的漲幅，周選擇權出現過+16000%的漲幅。

但波動大不代表風險大，重點在於投入資金相對總資產的比例。投資最大的風險在於人——如果你不知道自己在做什麼、自己憑什麼獲利，那你就承受了巨大的風險。

選擇權也是重要的避險商品，可用很小的資金達到避險效果。
