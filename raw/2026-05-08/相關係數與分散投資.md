## 相關係數（Correlation Coefficient）

相關係數是用以反映變數之間相關關係密切程度的統計指標，由卡爾·皮爾遜設計，又稱皮爾遜相關係數。

### 定義與分類

1. **簡單相關係數**：度量兩個變數間的線性關係，一般用 ρ 表示
2. **復相關係數**（多重相關係數）：因變數與多個自變數之間的相關關係
3. **典型相關係數**：對各組變數進行主成分分析後，再計算綜合指標間的相關

### 性質

- 相關係數取值範圍 -1 ≤ ρ ≤ +1
- ρ > 0：正相關，ρ 越接近 1 越強
- ρ < 0：負相關，ρ 越接近 -1 越強
- ρ = 0：不相關（但可能存在非線性關係）
- |ρ| > 0.8：高度相關
- |ρ| < 0.3：低度相關
- 其他為中度相關

### 重要注意事項

- **不相關 ≠ 獨立**：不相關僅指沒有線性關係，但可能存在非線性關係
- **樣本量影響**：樣本量 n 較小時，相關係數的絕對值易接近 1，產生假象；n 較大時，絕對值容易偏小
- **n=2 時**：相關係數的絕對值總為 1，毫無統計意義

### 實例：深滬股市

研究深滬兩市資產負債率與每股收益的相關關係：
- 資產負債率前40名上市公司：r = -0.6139（顯著負相關）
- 資產負債率後20名：r = 0.1072（弱正相關）
- 全部上市公司：r滬 = -0.5509，r深 = -0.4361

僅憑 r 的計算值大小判斷相關程度有一定缺陷，需結合樣本量與顯著性檢驗。

## 分散投資（Diversification）

分散投資就是要將資金分散地投入不同的行業、企業、市場等，以避開投資風險，追求投資收益最大化。

### 七種分散方式

1. **分散行業種類**：不集中購買同一行業股票，避免行業景氣衰退導致全面虧損
2. **分散企業單位**：投入多家企業，避免單一企業經營不善
3. **分散投資品種**：股票與債券結合，發現新的利潤點
4. **分散地域**：投資不同地區，降低地方法規與經營環境風險
5. **分散時間**：不同時間投入資金，避免生產周期與淡旺季影響
6. **分散板塊**：同時關注兩種以上不同特性的板塊
7. **分散市場**：不同市場的升跌不會同時發生，降低相關係數

### 相關係數與分散效果的關係

不同市場之間的不劃一因素使彼此股價的相關係數減低，這解釋了國際股票組合的波幅比單一國家為低的原因。

- 相關係數越低，分散效果越好
- 相關係數 = 1 時，分散投資無法降低風險
- 相關係數 = -1 時，理論上可完全消除風險
- 實務中，多數股票相關係數在 0.3~0.7 之間

### 投資組合風險公式

投資組合的變異數：
σ²p = w²A·σ²A + w²B·σ²B + 2·wA·wB·σA·σB·ρAB

當 ρAB < 1 時，組合風險低於加權平均風險，這就是分散投資的數學基礎。
