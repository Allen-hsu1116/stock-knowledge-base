## TEJ：市場恐慌還是機會？從融資維持率看穿轉折訊號

融資維持率（Maintenance Margin Ratio）是市場上用來衡量融資賬戶風險狀況的重要指標，可以用來判斷投資者是否接近券商發出追繳通知的門檻。

市場普遍認為融資維持率落在 130%～160% 較為安全；若低於130%，券商會給予投資人T+2的時間來補繳保證金、提供擔保品或減少融資部位，否則有權強制賣出持股（斷頭）。

### 融資維持率計算

融資維持率 = 名目股價 / (每股融資成本 × 融資乘數) × 100%

融資乘數：上市公司60%、上櫃公司50%

例子：
- 以每股100元買進，融資乘數60%
- 原融資維持率 = 100 / (100 × 60%) = 166%
- 股價跌至78元：維持率 = 78 / 60 = 130%（追繳門檻）
- 股價跌至60元：維持率 = 60 / 60 = 100%（斷頭風險）

### 為何出現極低維持率？

1. **高檔融資進場**：散戶常在股價創高時追高，導致平均融資成本偏高
2. **股價急跌**：如 AI、航運、重電類股修正時，帳戶價值迅速縮水
3. **未及時補繳保證金**：部分投資人選擇觀望不補繳，或無能力補繳
4. **市場氣氛恐慌**：引發連環斷頭效應，造成融資餘額與維持率惡化

### 時事案例（2025年4月）

台股單日下跌2,065點，跌幅9.6%，創史上第三大單日跌點。融資維持率下降至133.8%，低於130%的個股數量高達589檔，形成連環追繳潮。

- 緯創（3231）：TEJ融資維持率曾下探至91%
- 技嘉（2376）：維持率跌破100%，引發多日追繳賣壓

### 回測策略

TEJ回測了融資維持率異常偏低時的反轉策略：
- 進場條件：融資維持率 < 過去10日移動平均 + 紅K + 融資餘額 > 前5日平均×0.95
- 持有15個交易日，停利+40%，停損-10%
- 結果：年化報酬率18.39%，夏普值1.04，最大回撤-28.47%

## StockFeel：Margin Call 與 Force Sell

小李案例：以每股100元融資買進A生技股，自備4萬、融資6萬。
- 原融資維持率 = 10萬/6萬 = 166.67%
- 股價跌至67元：維持率 = 6.7萬/6萬 = 111.67% < 120%（追繳門檻）
- 小李需補1萬（融資金額從6萬降至5萬），維持率回升至6.7萬/5萬 = 134%
- 若不補繳，券商兩天後強制賣出（Force Sell/斷頭）

惡性循環：券商以低於市價賣出 → 拉低成交金額 → 融資維持率再降 → 新的 Margin Call → 再度 Force Sell

## Money101：收到融資追繳令怎麼辦？

### 融資使用時機
1. 資金不夠時，透過融資小變大（自備40%借60%）
2. 短期交易放大槓桿
3. 信用交易當沖（資買券賣）

### 融資維持率計算
融資維持率 = 股票現值 / 融資金額

整戶擔保維持率 = (融資擔保證券市值 + 原融券擔保價款及保證金) / (原融資金融 + 融券證券市值) × 100%

### 避免斷頭三階段
1. **T+2日內補到166%以上**：券商撤銷追繳令
2. **T+2日內補到130%～166%**：券商暫不強制處分，但追繳令仍存在；若再跌破130%，當日下午須立即補錢
3. **T+3日仍未補到130%以上**：券商強制拋售（斷頭）

### 台積電實例
以815元買進一張，融資48.9萬、自備32.6萬
- 初始維持率 = 81.5/48.9 = 166%
- 股價跌至635元以下（約78折）→ 維持率 < 130% → 追繳令

## 市場先生：維持保證金完整教學

### 三種保證金

| 類型 | 定義 |
|------|------|
| 原始保證金 | 進場前必須準備的最低金額 |
| 維持保證金 | 持倉期間帳戶必須維持的最低金額 |
| 追繳保證金 | 原始保證金與帳戶餘額的差額 |

### 台股融資 vs 期貨保證金比較

| 項目 | 台股融資 | 台灣期貨 |
|------|----------|----------|
| 原始保證金 | 股價40%-50% | 以期交所規定 |
| 維持保證金 | 融資維持率130% | 以期交所規定 |
| 追繳通知 | 低於維持率130% | 盤中/盤後低於維持保證金 |
| 補繳門檻 | 補至維持率166%以上 | 補至原始保證金 |
| 斷頭門檻 | T+3強制賣出 | 盤中風險指標<25%立即強平 |
| 加收保證金 | 無 | 單一商品超過20%加收20% |

### 關鍵建議
- 不要只放原始保證金，最好放相當於合約價值100%的保證金
- 保證金只是「最低門檻」，在最低標準操作代表在危險邊緣試探
- 控制好風險，沒那麼多錢就不要操作那麼大的部位
