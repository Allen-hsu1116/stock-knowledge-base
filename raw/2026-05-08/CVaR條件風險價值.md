# CVaR條件風險價值

**來源**：https://wiki.mbalib.com/zh-tw/CVaR
**日期**：2026-05-08
**主題**：風險管理

---

CVaR即條件風險價值（Conditional Value at Risk），是由Rockafellar和Uryasev於1997年提出的一種較VaR更優的風險計量技術，其含義為在投資組合的損失超過某個給定VaR值的條件下，該投資組合的平均損失值。

若設定投資組合的隨機損失為-X(-X<0)，VaRβ是置信水平為1-β的VaR值，則CVaR可用數學公式表示為：

CVaRβ=E(-X|-X≥VaRβ)

CVaR是在VaR的基礎上發展出來的一種投資風險計量方法。VaR作為風險計量方法不僅具有概念簡單、易於溝通和理解的優點，而且為不同金融工具構成的複雜的投資組合提供了一個統一的、綜合性的風險測量框架。因此，VaR現在被廣泛應用於各金融機構，並且正在成為計量金融風險的國際標準。然而，許多實證研究表明，VaR方法具有其本身無法克服的缺陷：

1. VaR不滿足一致性公理，這就意味著用VaR來計量風險，投資組合的風險不一定小於或等於該組合中各種資產分別計量的風險值之和，這與風險分散化的市場現象相違背
2. VaR尾部損失測量的非充分性，它無法考察超過分位點的下方風險信息
3. VaR應用的前提必須是股票收益率服從正態分佈，而許多實證研究表明股票收益率並不服從正態分佈

為了克服VaR的缺陷，Rockafellar和Uryasev在2000年提出了CVaR的風險計量技術。CVaR是指投資組合的損失大於某個給定的VaR值的條件下，該投資組合損失的平均值。與VaR相比，CVaR滿足次可加性、正齊次性、單調性及傳遞不變性，因而CVaR是一種一致性的風險計量方法。另外，研究表明，CVaR可以通過使用線性規劃算法來進行優化。

## CVaR的性質

1. 平移不變性：對於任意一個固定的常數c，有Cβ(Y + c) = Cβ(Y) + c
2. 正齊次性：對於任意正數c，有Cβ(cY) = cCβ(Y)
3. 單調可加性
4. 某種程度上具有關於零的對稱性
5. CVaR具有次可加性：若0<λ<1，對任意兩個損失變數Y1和Y2有Cβ(λY1+(1-λ)Y2)≤λCβ(Y1)+(1-λ)Cβ(Y2)

因此無論投資組合的回報是否是正態分佈，CVaR都是一致性風險度量。此外，CVaR是凸性的風險計量，因此基於CVaR的投資組合優化必定存在最小風險的解，而VaR並不是一個凸性的風險計量，可能不存在最優的解。

## CVaR的優缺點

### 優點
1. 考慮了超過VaR的尾部風險
2. 屬於一致風險測度，滿足了可加性
3. 不易被操縱，不易出現誤導投資者的信息
4. CVaR的凸性使得基於CVaR的投資組合優化更易於實施

### 缺點
1. 不與三階及三階以上的隨機占優相一致
2. 不與公司的違約概率直接相聯繫
3. 不易於實施有效的事後檢驗
4. CVaR與分佈的尾部估計關係密切，因而不能確保估計結果的穩定性

## CVaR的應用

### 投資組合的優化
經典投資組合模型是以最小方差為基準目標的，以此構成了馬柯威茨的有效邊界。標準差並不能告訴投資者投資組合的潛在損失到底是多大。因此在此加入CVaR對投資組合進行約束，在均值-方差空間上表現為一條與馬柯威茨邊界相交的斜線，通過頭寸的調整來預先限定組合的潛在風險。

### 確定內部風險資本需求和設定風險限額
利用CVaR可以確定金融機構在整體上為抵禦市場風險所需求的內部風險資本，並為交易員或業務部門設置風險限額，以防止過度投機行為。CVaR限額考慮了組合的風險分散的效果，並且有助於設置層次性限額結構。

### 資本配置
在經典的投資組合模型中以最小方差為規劃目標，用CVaR來代替方差，以最小CVaR為規劃目標，即構成最優均值-CVaR模型。在收益一定時，通過調整頭寸使CVaR最小；在CVaR一定時，通過調整頭寸使收益最大化。
