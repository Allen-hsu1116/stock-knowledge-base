# 錨定效應 Anchoring Bias

## 概述

錨定效應（Anchoring Effect），又稱沉錨效應、錨定陷阱，是一種重要的心理現象。指人們在對某個事件做定量估測時，會將某些特定數值作為起始值，起始值像錨一樣制約著估測值。做決策時，會不自覺地給予最初獲得的信息過多重視。

## 核心內容

### 經典實驗（Kahneman & Tversky, 1974）

要求實驗者估計非洲國家在聯合國所占席位百分比。先旋轉羅盤隨機選擇0-100之間的數字，再向下或向上調整估計。結果：隨機選定10和65的兩組，平均估計值分別為25和45。即使知道數字是隨機的，人們仍被「錨定」。

### 錨定效應的形成條件

1. **參考物引起足夠注意**：只要「錨」被注意到，無論多荒謬都會產生影響
2. **參照物與目標的相似性**：單位相同時效應最強（金額→金額，百分比→百分比）
3. **極端型參照物**：即使極端值仍會產生錨定效應，只是較弱
4. **認知能力無法消除**：即使被告知不要受影響，錨定效應仍然存在
5. **獎勵無效**：物質或精神獎勵都無法顯著減少錨定現象

### 錨定效應的類型

1. **經典錨定效應**（語意啟動範式）：先比較後估計的兩步範式
2. **基礎錨定效應**（數字啟動範式）：單純數字呈現就能產生影響
3. **潛意識錨效應**（閾下啟動範式）：即使潛意識感知也會受影響
4. **自生錨 vs 實驗者錨**：自生錨可透過努力思考減少影響，實驗者錨幾乎不可控

### 心理機制

- **選擇通達模型（SA）**：假設錨值即為目標值，積極搜索支持證據，導致偏差
- **不充分調整啟發式**：從錨值向目標值調整不充分，調整在「可接受邊界」就停止
- **雙重心理過程**：外部錨是自動化、無意識的；自生錨是有意識、可調整的

### 股市中的錨定效應

股票當前價格的確定受到過去價格影響，呈現錨定效應。在沒有更多信息時，過去的價格成為當前價格的重要決定因素。商品價值越模糊，參考價格就越重要，錨定效應就越強。

### 避免錨定陷阱的方法

1. **反向思維策略**：主動思考「為什麼這個錨值可能是不對的」
2. **類比的錨**：用類比估計替代直接從錨值調整
3. **多元化參考點**：不要只看單一錨點，多角度估算
4. **延遲判斷**：給自己時間冷靜，減少直覺反應的影響
