## 理財研究室 FM Studio — 艾略特波浪理論完整指南

### 核心基礎：掌握市場的心理節奏

艾略特波浪理論（Elliott Wave Theory）由美國會計師拉爾夫・納爾遜・艾略特（Ralph Nelson Elliott）在1930年代提出。他透過對75年道瓊工業平均指數的深入研究，發現看似混亂的股市波動，實際上遵循著可識別的、重複出現的自然規律。背後的驅動力是所有市場參與者共同形成的集體心理變化——從樂觀到悲觀，再從悲觀回歸樂觀的循環。

### 5-3波浪結構

- **推動浪 (Impulse Waves)**：由5個子浪組成，標示為1、2、3、4、5。方向與更大級別的主趨勢一致。
- **修正浪 (Corrective Waves)**：由3個子浪組成，標示為A、B、C。方向與主趨勢相反。

### 推動浪各波特徵

- **第1浪**：趨勢的開端。通常由少數嗅覺敏銳的投資者發動，常被誤認為熊市反彈。
- **第2浪**：獲利了結的回檔。回檔幅度通常較深，但絕不低於第1浪起點。市場情緒悲觀。
- **第3浪**：最長、最強勁的主升段。基本面好轉，大量投資者進場追價，成交量顯著放大。
- **第4浪**：橫向整理的修正波。走勢偏向橫盤，成交量萎縮，為最後一波做準備。
- **第5浪**：趨勢最後衝刺。散戶大量湧入，但動能已不如第3浪，常出現價量背離。

### 修正浪各波特徵

- **A浪**：修正的開始。市場情緒仍樂觀，許多人視為買入良機。
- **B浪**：假性反彈（死貓跳）。讓投資者誤以為牛市將繼續，是多頭陷阱。
- **C浪**：修正的最後一段。市場信心崩潰，恐慌性拋售，C浪通常具5浪結構。

### 三大鐵律

1. **鐵律一**：第2浪的回檔低點，永遠不能低於第1浪的起點。
2. **鐵律二**：在三個推動浪（1、3、5）中，第3浪永遠不能是最短的一個。
3. **鐵律三**：第4浪的價格低點，永遠不能進入第1浪的價格範圍（不可重疊）。（斜向三角形除外）

### 碎形結構（Fractal Nature）

任何一個波浪都可以被細分成更小級別的波浪，這些小波浪也遵循同樣的5-3結構。這使得波浪理論可以應用於不同時間框架，從超級大循環浪到微浪都適用。

### 波浪與斐波那契的結合

| 波浪關係 | 常見斐波那契比例 | 實戰意涵 |
|----------|------------------|----------|
| 第2浪回檔 | 第1浪長度的 50%, 61.8%, 78.6% | 潛在買入點 |
| 第3浪延伸 | 第1浪長度的 161.8%, 261.8%, 423.6% | 預測第3浪目標價位 |
| 第4浪回檔 | 第3浪長度的 23.6%, 38.2%, 50% | 可能的支撐區 |
| 第5浪目標 | 與第1浪等長 (100%) 或第1-3浪總長的 61.8% | 預估第5浪終點 |

### 修正浪三大形態

1. **鋸齒形 (Zigzag)**：5-3-5結構，走勢陡峭，回檔幅度大，通常出現在第2浪。
2. **平台形 (Flat)**：3-3-5結構，橫向整理，顯示主趨勢依然強勁，通常出現在第4浪。
3. **三角形 (Triangle)**：3-3-3-3-3結構，能量收縮的盤整形態，幾乎只出現在第4浪或B浪。

## 投資兄弟 — 艾略特波浪理論8大波浪定律

### 波浪特性與投資者心理

| 波浪 | 特性與市場情緒 |
|------|----------------|
| 波浪1 | 趨勢開端，通常不明顯。市場仍被負面消息籠罩，僅少數敏銳投資者開始佈局 |
| 波浪2 | 對第1波的回檔，但永不跌破第1波起點。市場情緒悲觀，成交量萎縮 |
| 波浪3 | 通常最強勁、最長的主升浪，價格迅速上漲。基本面好轉，大量資金入場 |
| 波浪4 | 複雜的橫盤整理或回檔，低點不應低於第1波高點。獲利了結賣壓出現 |
| 波浪5 | 趨勢最後一波，市場情緒達到極度樂觀。成交量可能不如第3波，技術指標可能背離 |
| 波浪A | 調整開始，下跌常被視為牛市正常回檔 |
| 波浪B | 假反彈/死貓跳，無法創出新高 |
| 波浪C | 調整最後一波，最具殺傷力。恐慌情緒蔓延，價格大幅下跌 |

### 延伸波與「邪惡第五波」

- 延伸波通常只會在一個推動浪中出現一次
- 最常見的延伸波發生在第3波，代表市場趨勢極其強勁
- **邪惡第五波**：延伸波出現在第5波時，市場情緒達到巔峰，各種利好消息滿天飛，吸引大量散戶追高。隨後的ABC調整波跌幅往往吞噬第5波大部分甚至全部漲幅。

### 實戰交易策略

1. **時機一（較激進）**：在判斷第2波回檔結束、價格準備突破第1波高點時進場。利潤空間最大，但風險也較高。
2. **時機二（較穩健）**：在第4波整理結束、價格突破第3波高點時進場。趨勢明確但利潤空間較小。

### 風險管理

- 主觀性是最大挑戰：十個分析師可能有十一種數浪方法
- 切勿單獨使用：建議與RSI、MACD、成交量等指標結合
- 嚴格設定止損：若第2波跌破第1波起點，則牛市假設失效

### 適用範圍

- 適用：大規模市場（指數、外匯、期貨），流動性高、參與者眾多的市場
- 不適用：股本小、易受人為操控的個股
- 中長期分析優於短線日內交易
