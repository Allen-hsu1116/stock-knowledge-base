## SMA 指標原理

SMA 均線是一種落後指標，透過計算過去特定時間內商品價格的平均值，來獲取市場趨勢資訊。這個平均值可以作為基準點，與當前價格及歷史平均價格做比較，判斷市場處於上漲趨勢、下跌趨勢或震盪趨勢。SMA 可以去除市場短期價格雜訊。

## SMA 指標公式

SMA = (P1 + P2 + P3 + ... + Pn) / n

P 為每日收盤價，n = 時間

## SMA 指標優缺點

**優點**：
- 容易理解，不需深厚數理背景
- 廣泛應用於股票、期貨、加密貨幣等市場
- 可根據交易習慣套用不同時間週期
- 快速反應價格變化趨勢

**缺點**：
- 落後指標，無法預測未來行情
- 劇烈震盪時信號延遲
- 不適合短線交易參考

## 均線扣抵

扣抵用來預測均線未來趨勢。捨去最早的收盤價，加入最新的收盤價：
- 最新收盤價 < 最舊收盤價 → 扣抵值下降 → 均線往下走
- 最新收盤價 > 最舊收盤價 → 扣抵值上升 → 均線往上走

## SMA 指標應用方法

- **判斷趨勢**：價格在 SMA 均線之上＝上漲趨勢；之下＝下跌趨勢
- **判斷買賣信號**：價格由下向上穿過 SMA → 買進；由上向下穿過 SMA → 賣出

## 多頭排列與空頭排列

**多頭排列**：短期、中期到長期的移動平均線依次向上排列，每一條均線都位於前一條均線的上方，形成「階梯式」向上趨勢。反映市場普遍處於樂觀情緒，暗示持續上升趨勢。

**空頭排列**：短期、中期到長期的移動平均線依次向下排列，每一條均線都位於前一條均線的下方，形成「階梯式」向下趨勢。代表市場普遍處於悲觀情緒，暗示持續下降趨勢。

## 多頭排列選股實戰用法

常見判斷方式採用 3 條均線：
- 5MA、20MA、60MA
- 7MA、30MA、90MA

**注意**：短中長期均線相差倍數最好在 2-5 倍之間比較有顯著效果。

**操作策略**：
- 均線完成多頭排列 + 市場價格在短期均線之上 → 考慮買入
- 均線開始失去多頭排列結構，或價格跌破關鍵均線（如 20MA）→ 考慮賣出或減倉

## 策略邏輯

- 雙均線黃金交叉與死亡交叉策略
- 黃金交叉買進做多
- 死亡交叉賣出做空
- 搭配止損止盈

建議搭配其他技術分析指標（如 RSI），不要過度依賴單一指標。
