# TRIX三重指數平滑移動平均指標教學

**來源**：https://www.taindicators.com/trix/
**日期**：2026-05-17
**主題**：技術分析

---

Triple Exponential (TRIX) 三重指數平滑移動平均指標是根據移動平均線理論，對數據進行三次平滑處理，再根據這條移動平均線的變動情況來預測股價的長期走勢。

## 計算方法

1. 首先計算 N 天收盤價的指數移動平均數值 AX
2. 然後計算 N 天的 AX 的指數移動平均數值 BX
3. 然後計算 N 天的 BX 的指數移動平均數值 CX（至此已經對 N 天的收盤價進行了三次平滑處理）
4. TRIX = (當日CX - 前一日CX) / 前一日CX

## 應用例子

### TRIX線／信號線交叉

TRIX 常與一條TRIX的移動平均線(如9天)一起使用:

- 當TRIX線從下向上突破信號線，預示著股價開始進入強勢拉升階段，投資者應及時買進股票。
- 當TRIX線向上突破TRMA線後，TRIX線和信號線同時向上運動時，預示著股價強勢依舊，投資者應堅決持股待漲。
- 當TRIX線在高位向下突破信號線，預示著股價強勢上漲行情已經結束，投資者應賣出餘下股票，及時離場觀望。
- 當TRIX線向下突破信號線後，TRIX線和信號線同時向下運動時，預示著股價弱勢特徵依舊，投資者應堅決持幣觀望。
- TRIX指標不適用於對股價的盤整行情的研判。

### 背馳訊號

- 頂背離：當股價K線圖上的股票走勢一峰比一峰高，股價在一直向上漲，而TRIX指標圖上的TRIX曲線的走勢是在高位一峰比一峰低，這叫頂背離現象。頂背離現象一般是股價將高位反轉的信號，表明股價短期內即將下跌，是比較強烈的賣出信號。
- 底背離：當股價K線圖上的股票走勢一峰比一峰低，股價在向下跌，而TRIX指標圖上的TRIX曲線的走勢是在低位一底比一底高，這叫底背離現象。底背離現象一般是股價將低位反轉的信號，表明股價短期內即將上漲，是比較強烈的買入信號。

指標背離一般出現在強勢行情中比較可靠。即股價在高位時，通常只需出現一次頂背離的形態即可確認行情的頂部反轉，而股價在低位時，一般要反覆出現多次底背離後才可確認行情的底部反轉。
