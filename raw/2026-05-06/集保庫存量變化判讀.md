# 集保庫存量變化判讀：看懂籌碼流向的隱形指標

**來源**：自行整理（基於公開資訊觀測站、台灣集保結算所資料）
**日期**：2026-05-06
**主題**：籌碼面分析

## 什麼是集保庫存量？

集保庫存量是指存放在台灣集中保管結算所（TDCC）的股票總數量，也就是市場上可以自由流通的股票數量。當公司股票上市或透過私募、員工庫藏股等方式移出集保時，庫存量會減少；反之，當大股東將持股轉存入集保、或員工執行認股權時，庫存量會增加。

## 集保庫存量變化的意義

### 庫存量減少（籌碼減少）

1. **大股東持股移出集保**：大股東將股票從集保帳戶提領，通常有以下可能：
   - 準備質押借款
   - 準備賣出（但不一定立即賣）
   - 移轉至其他保管機構
   - 參與私募或策略性移轉

2. **庫藏股註銷**：公司買回庫藏股後辦理減資註銷，集保庫存量會減少，流通股數下降，EPS提升

3. **私募或策略性鎖定**：特定人認購私募股票後鎖定不流通，降低市場浮動籌碼

### 庫存量增加（籌碼增加）

1. **大股東轉存入集保**：通常準備賣出，是潛在賣壓訊號
2. **員工認股權執行**：員工行使認股權後存入集保，增加流通籌碼
3. **現金增資新股上市**：新發行股票存入集保，籌碼稀釋

## 實戰判讀重點

### 1. 庫存量變化率比絕對值重要

單看庫存量絕對值意義有限，重點是觀察變化趨勢：
- 連續多週庫存量減少 → 籌碼正在集中
- 連續多週庫存量增加 → 籌碼正在分散

### 2. 搭配股價位置判斷

- **低檔區庫存量減少**：大戶在低檔收籌碼，偏多訊號
- **高檔區庫存量增加**：大股東可能準備出貨，偏空訊號
- **低檔區庫存量增加**：可能認股權執行或現增，需看後續賣壓
- **高檔區庫存量減少**：可能私募鎖定或庫藏股註銷，需確認原因

### 3. 與股權分散表交叉驗證

集保庫存量變化要搭配股權分散表（千張大戶持股比例變化）一起看：
- 庫存量減少 + 千張大戶增加 → 強烈偏多訊號
- 庫存量增加 + 千張大戶減少 → 強烈偏空訊號
- 庫存量減少 + 千張大戶減少 → 可能是大股東移出集保準備質押，中性偏空
- 庫存量增加 + 千張大戶增加 → 可能是法人或大戶從場外買入，中性偏多

### 4. 特殊事件對庫存量的影響

- **除權息前**：可能出現棄息賣壓，庫存量暫時性變化
- **減資前**：庫存量依比例減少，這是機械性變化不是籌碼訊號
- **合併換股**：庫存量會有較大波動，需排除一次性因素
- **ETF申購贖回**：大型ETF調整成分股時會影響相關個股庫存量

## 計算公式

- 庫存量變化率 = （本期庫存量 - 上期庫存量）÷ 上期庫存量 × 100%
- 流通比率 = 集保庫存量 ÷ 發行股數 × 100%
- 籌碼集中度 = 千張大戶持股 ÷ 集保庫存量 × 100%

## 注意事項

1. **集保庫存量有延遲性**：通常落後1-2週，不能當即時指標使用
2. **大股東移出集保≠立即賣出**：移出只是準備動作，真正賣出要看內部人申報轉讓
3. **季節性因素**：員工分紅旺季（通常Q1）、現增季（通常Q3-Q4）會有例行性變化
4. **合併與減資需排除**：這些是機械性變動，不反映籌碼意願
5. **搭配其他籌碼指標**：單看庫存量變化容易誤判，必須搭配三大法人買賣超、融資融券變化、分點進出交叉驗證
