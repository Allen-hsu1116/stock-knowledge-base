## 選擇權Greeks是什麼

Greeks是選擇權定價模型（Black-Scholes）中，衡量選擇權價格對各因素敏感度的指標。每個Greek代表一個風險維度，理解Greeks才能知道你到底在承擔什麼風險。

四大Greeks：
- **Delta（δ）**：方向風險
- **Gamma（γ）**：加速度風險
- **Theta（θ）**：時間衰變風險
- **Vega（ν）**：波動率風險

## Delta（δ）— 方向風險

### 定義
標的價格每變動1元，選擇權價格變動多少。

- 買權Delta：0到+1
- 賣權Delta：-1到0
- 平值買權Delta ≈ 0.5
- 深價內買權Delta → 1
- 深價外買權Delta → 0

### Delta的實戰意義

1. **相當於持有多少張股票**：Delta=0.5的買權，相當於持有0.5張股票的方向暴露
2. **機率近似值**：Delta ≈ 選擇權到期時進入價內的機率。Delta=0.3約有30%機率到期有價值
3. **避險比例**：持有1張股票，買Delta=-1的賣權可以完全對沖方向風險

### Delta中性策略
讓組合總Delta=0，不賺方向只賺波動率或時間價值。

範例：
- 買1口Delta=0.5的Call（總Delta=+0.5）
- 賣出50股股票（總Delta=-0.5）
- 組合Delta = 0.5 - 0.5 = 0

### Delta隨價格變化
- 價外→價內時Delta從0往1走（買權）或0往-1走（賣權）
- 這就是Gamma的作用：Delta的變化率

## Gamma（γ）— 加速度風險

### 定義
標的價格每變動1元，Delta變動多少。Gamma是Delta的導數。

### Gamma的實戰意義

1. **Delta不穩定性**：Gamma越大，Delta變化越快，方向暴露變化越劇烈
2. **買方有正Gamma**：價格往有利方向走時Delta加速變大，往不利方向走時Delta加速變小——這是買方的優勢
3. **賣方有負Gamma**：價格往不利方向走時虧損加速——這是賣方的風險

### Gamma風險最大的時刻

- **平值附近**：Gamma最大，Delta變化最劇烈
- **接近到期**：Gamma急劇放大，短時間內Delta可能從0.1跳到0.9
- **Gamma Squeeze**：做市商必須避險，買入越漲越多的股票，正回饋循環推升股價

### Gamma Scalping
利用正Gamma，在價格波動中不斷調整Delta中性部位，低買高賣賺價差。

## Theta（θ）— 時間衰變風險

### 定義
每過一天，選擇權時間價值減少多少。Theta通常是負數（對買方而言）。

### Theta的實戰意義

1. **時間是買方的敵人**：每過一天，買方的選擇權就貶值一些
2. **時間是賣方的朋友**：賣方什麼都不做，時間就把時間價值送進口袋
3. **非線性衰變**：接近到期時Theta加速（時間價值像冰塊融化，越到最後融化越快）

### Theta衰變曲線

| 距到期天數 | 每日時間價值衰變（平值買權） |
|-----------|----------------------------|
| 90天 | 緩慢，約總價值的0.3%/天 |
| 60天 | 中等，約0.5%/天 |
| 30天 | 加速，約1%/天 |
| 7天 | 急劇，約3-5%/天 |
| 1天 | 最後一滴，Theta接近全部剩餘時間價值 |

### Theta與Gamma的取捨

這是一體兩面：
- **買方**：正Gamma（賺加速）+ 負Theta（賠時間）→ 需要波動夠大來彌補時間衰變
- **賣方**：負Gamma（賠加速）+ 正Theta（賺時間）→ 需要波動夠小來保住時間收入

**核心公式**：Theta損失 ≈ 0.5 × Gamma × (σ²) × S²

這就是為什麼高波動時賣方雖然Theta收入高，但Gamma風險也大。

### 三種Theta策略

1. **賣出寬跨式（Iron Condor）**：穩定Theta收入，但Gamma風險大
2. **日曆價差（Calendar Spread）**：近月Theta衰變快、遠月慢，賺時間差
3. **Covered Call**：持有股票+賣出買權，降低持倉成本但限制上漲空間

## Vega（ν）— 波動率風險

### 定義
隱含波動率（IV）每變動1個百分點，選擇權價格變動多少。

### Vega的實戰意義

1. **買方有正Vega**：IV上升時買方獲利，IV下降時買方虧損
2. **賣方有負Vega**：IV下降時賣方獲利，IV上升時賣方虧損
3. **Vega在平值最大**：越接近價平，對波動率變化越敏感
4. **遠月Vega更大**：長期選擇權對波動率變化更敏感

### IV Crush（波動率崩塌）

財報公布、重大事件結束後，IV急劇下降，選擇權買方即使方向對也可能虧錢。

範例：
- 財報前IV=50%，買入Call
- 財報後IV降到30%
- 即使股價小漲，Vega損失可能大於Delta收益

### Vega與Theta的關係

高IV → 高Vega + 高Theta
- 買方：高Theta損失需要大波動彌補
- 賣方：高Theta收入但Vega風險也大

低IV → 低Vega + 低Theta
- 買方：Theta損失小，但需要波動啟動
- 賣方：Theta收入少，但Vega風險也小

## Greeks整合判讀

### 買方 vs 賣方 Greeks對比

| Greek | 買方（Long） | 賣方（Short） |
|-------|-------------|-------------|
| Delta | 方向一致（+Call/-Put） | 方向相反（-Call/+Put） |
| Gamma | 正（有利） | 負（不利） |
| Theta | 負（不利） | 正（有利） |
| Vega | 正（賭IV上升） | 負（賭IV下降） |

### 四象限策略選擇

| 預期 | 策略 | Delta | Vega |
|------|------|-------|------|
| 看大漲 | 買Call | + | + |
| 看小漲 | 賣Put | + | - |
| 看大跌 | 買Put | - | + |
| 看小跌 | 賣Call | - | - |
| 看盤整 | Iron Condor | ≈0 | - |
| 看大波 | 買Straddle | ≈0 | + |

### 風險管理矩陣

| 風險類型 | 對應Greek | 管理方法 |
|----------|----------|---------|
| 方向風險 | Delta | Delta中性避險 |
| 加速度風險 | Gamma | 限制單一部位Gamma暴露 |
| 時間衰變 | Theta | 選擇適當到期日，避免最後一週 |
| 波動率風險 | Vega | 分散到期日，控制IV暴露 |
| 利率風險 | Rho | 長期LEAPS需考慮，短線可忽略 |

## 注意事項

1. **Greeks是瞬時指標**：只反映當前參數下的敏感度，標的價格一動Greeks就跟著變

2. **到期日效應**：接近到期時Greeks變化劇烈，最後一週Theta和Gamma都會急劇放大

3. **IV不是歷史波動率**：Vega衡量的是IV變動的影響，IV包含市場預期溢價，可能偏離HV很遠

4. **模型風險**：Black-Scholes假設波動率恆定、報酬率常態分配，這些假設在極端市況下失效

5. **實務上用絕對值管理**：組合的淨Greeks要用絕對值控制，避免單一方向暴露過大
