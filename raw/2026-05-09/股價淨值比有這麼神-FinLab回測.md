# 股價淨值比有這麼神？

**來源**：https://www.finlab.tw/%E8%82%A1%E5%83%B9%E6%B7%A8%E5%80%BC%E6%AF%94%E6%9C%89%E9%80%99%E9%BA%BC%E7%A5%9E%EF%BC%9F/
**日期**：2026-05-09
**主題**：基本面分析

---

股價淨值比(PB)，是一個用來判斷究竟公司是被高估還是低估的指標。

FinLab 回測實驗（2010-2017年台股）：
- 每半年（125天）換一次持股
- 選取所有 PB < N 的公司，資產平均分配，持有半年
- 結論：PB 越小，整體報酬率越高

但 PB < 0.5 時有問題：
- 回測結果大起大落
- 2015-2016 年持股數量極少（2-3 檔），風險集中
- 最大回撤達 40%
- 持股數量與大盤成反比（大盤漲→PB小股票少→大盤跌→PB小股票多）

結論：
- 選擇 PB < 0.6~0.8 的一群標的，半年更新一次還不錯
- 一定要做好風險管理，千萬別將全部的雞蛋放在同一個籃子裡
- 可以配合其它財報選出優質股
