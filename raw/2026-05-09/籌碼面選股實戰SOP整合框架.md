# 籌碼面選股實戰SOP：從零散指標到可執行框架

**來源**：知識庫整合 + 實戰經驗總結
**日期**：2026-05-09
**主題**：籌碼面分析

---

## 一、籌碼面選股的核心邏輯

籌碼面選股的核心不是看單一指標，而是看「誰在買、買多少、買多久」。

三個核心問題：
1. **主力進場了嗎？** → 看分點、法人、融資券
2. **主力還在嗎？** → 看持股集中度、庫存量變化
3. **主力何時出場？** → 看出貨訊號、量價變化

---

## 二、籌碼面選股七步SOP

### Step 1：三大法人方向確認

**目的**：確認大方向，法人方向是風向球

檢查項目：
- **外資**：連續買超5日以上偏多，連續賣超5日以上偏空
- **投信**：連續買超3日以上開始關注，10日以上強烈偏多
- **自營商**：避險買超要扣掉（隔天大概率賣出），自行買超才有意義

**過濾條件**：
- 外資買超 + 投信買超 = 最強組合
- 外資賣超 + 投信買超 = 投信獨撐，力道有限
- 三大法人全買 = 共振最強
- 三大法人全賣 = 絕對避開

**注意**：三大法人訊號可能失真（換倉、避險、結算），需要交叉判讀期貨和選擇權部位

### Step 2：主力分點追蹤

**目的**：找出真正的主力是誰

檢查項目：
- **買超集中度**：前5大買超分點占總買超超過50%，主力明確
- **地緣券商**：公司附近分點大量買超，可能是內部人或公司派進場
- **神秘券商**：連續多日出現在同一檔股票的陌生分點，可能是主力分單
- **隔日沖券商**：凱基、元大、國票、富國等隔日沖大戶，看到要避開或短線操作

**過濾條件**：
- 買超集中度 > 50%：主力明確
- 買超集中度 < 30%：散戶為主，難有大行情
- 主力勝率 > 60%：值得跟單
- 主力勝率 < 40%：主力也在賠，避開

### Step 3：融資券動態判讀

**目的**：判斷散戶與主力對決態勢

檢查項目：
- **融資餘額變化**：增加代表散戶看多，減少代表散戶看空或停損
- **融券餘額變化**：增加代表看空力量增強
- **券資比**：>30% 表示融券偏重，有軋空行情可能
- **融資現償 vs 融券現償**：同時出現大量代表多空對決加劇

**八種組合判讀**：
| 融資 | 融券 | 價格 | 判讀 |
|------|------|------|------|
| 增 | 增 | 漲 | 主力與散戶同步看多，偏多 |
| 增 | 減 | 漲 | 多方強勢，空方認輸回補 |
| 減 | 增 | 漲 | 主力吸貨散戶看空，最偏多 |
| 減 | 減 | 漲 | 縮量上漲，籌碼穩定 |
| 增 | 增 | 跌 | 多空對決激烈，方向未明 |
| 增 | 減 | 跌 | 散戶多單停損，偏空 |
| 減 | 增 | 跌 | 主力出貨散戶看空，最偏空 |
| 減 | 減 | 跌 | 縮量下跌，賣壓漸歇 |

### Step 4：股權分散度確認

**目的**：確認籌碼是集中還是分散

檢查項目：
- **千張大戶持股比例**：增加代表籌碼集中，偏多
- **集保庫存量變化**：減少代表籌碼集中（鎖在實體），偏多
- **股東人數變化**：減少代表籌碼集中，增加代表籌碼分散

**過濾條件**：
- 千張大戶比例上升 + 股東人數下降 = 最佳籌碼組合
- 千張大戶比例下降 + 股東人數上升 = 主力出貨，避開

### Step 5：借券與外資期貨交叉驗證

**目的**：確認外資真實方向

檢查項目：
- **借券賣出餘額**：持續增加代表外資隱形做空
- **外資期貨未平倉**：空單增加代表看空，多單增加代表看多
- **外資現貨買賣超**：與期貨方向一致才可靠
- **選擇權PCR**：成交量PCR與未平倉PCR反向時是反轉訊號

**交叉判讀**：
- 現貨買超 + 期貨多單 = 真正看多
- 現貨買超 + 期貨空單 = 避險或套利，不是真看多
- 現貨賣超 + 期貨空單 = 真正看空
- 現貨賣超 + 借券增加 = 雙重看空

### Step 6：籌碼七指標共振確認

**目的**：多指標共振提高勝率

七個籌碼指標：
1. **三大法人連續買超**：外資+投信連續3日以上買超
2. **主力買超**：前5大分點買超占總買超>50%
3. **融資使用率下降**：散戶退場主力進場
4. **券資比上升**：融券增加有軋空潛力
5. **千張大戶持股增加**：大戶加碼
6. **集保庫存量減少**：籌碼集中
7. **內部人未申報轉讓**：公司派沒有賣股

**共振標準**：
- 3項以下：訊號弱，不進場
- 4-5項：訊號中等，可小量進場
- 6-7項：訊號強烈，可正常部位進場

### Step 7：進場時機與停損設定

**目的**：確認進場點位與風險控制

進場條件（全部滿足才進場）：
1. 籌碼七指標≥4項共振
2. 技術面有突破或支撐確認
3. 量價配合（帶量突破或量縮回測）
4. 大盤不在系統性風險區（VIX<25、外資未大量賣超）

停損設定：
- **籌碼停損**：主力買超連續3日轉為賣超
- **技術停損**：跌破進場支撐或關鍵均線
- **比例停損**：虧損達7-10%出場
- **時間停損**：進場後5日未如預期走勢出場

## 三、籌碼面選股常見錯誤

### 錯誤1：單一指標判斷
只看三大法人買超就進場，忽略避險部位和換倉影響。

**修正**：至少用三個以上指標交叉驗證。

### 錯誤2：忽略量價配合
籌碼面偏多但量價背離，主力可能在出貨。

**修正**：籌碼面+量價面+技術面三面合一。

### 錯誤3：追高隔日沖
看到隔日沖券商買超就追，結果隔天被倒貨。

**修正**：先辨識隔日沖券商分點，4%分水嶺原則。

### 錯誤4：融資增加就偏多
融資增加代表散戶進場，反而可能是主力出貨的訊號。

**修正**：融資增加要看股價位置，高檔增加偏空、低檔增加可能是底部吸貨。

### 錯誤5：忽略大盤環境
個股籌碼再好，大盤崩跌也會跟著跌。

**修正**：先判斷大盤多空，再選個股。

## 四、籌碼面選股效率工具

### 每日必查清單
1. 三大法人買賣超（現貨+期貨+選擇權）
2. 主力分點買賣超（前5大）
3. 融資券餘額變化
4. 集保股權分散表（每週）
5. 借券賣出餘額
6. 選擇權PCR

### 快速篩選條件
- 三大法人連續買超 ≥ 3日
- 主力買超集中度 ≥ 50%
- 融資使用率 < 30%
- 券資比 > 20%（有軋空潛力）
- 千張大戶持股比例上升

## 五、實戰案例框架

### 多頭籌碼組合
外資連續買超 + 投信連續買超 + 主力買超集中度>50% + 融資減少 + 千張大戶增加 + 借券餘額低 + 選擇權PCR偏多

### 空頭籌碼組合
外資連續賣超 + 投信連續賣超 + 主力賣超集中度>50% + 融資增加 + 千張大戶減少 + 借券餘額增加 + 選擇權PCR偏空

### 軋空行情組合
券資比>50% + 融券餘額持續增加 + 接近停止過戶日（強制回補）+ 主力鎖碼 + 量縮不跌
