## 什麼是市場體制

市場體制（Market Regime）是指市場在某一段時間內呈現的特徵行為模式。不同體制下，同一種策略的表現可能天差地遠。判斷當前市場處於哪種體制，是選擇正確策略的第一步。

## 四種主要市場體制

### 1. 趨勢上升體制（Trending Up）
- 價格持續創新高，低點也逐步抬高
- 均線多頭排列，ADX > 25
- 適用策略：順勢交易、突破買進、均線回測買進
- 最佳指標：移動平均線、SAR、ADX

### 2. 趨勢下降體制（Trending Down）
- 價格持續創新低，高點也逐步下降
- 均線空頭排列，ADX > 25
- 適用策略：放空、順勢賣出、反彈做空
- 最佳指標：移動平均線、SAR、ADX

### 3. 震盪體制（Range-bound / Mean-reverting）
- 價格在支撐壓力區間內反覆震盪
- ADX < 20，均線糾結
- 適用策略：區間交易、均值回歸、網格交易
- 最佳指標：RSI、KD、布林通道、支撐壓力

### 4. 高波動混亂體制（Volatile / Choppy）
- 價格劇烈波動但無明確方向
- ATR異常放大，頻繁假突破
- 適用策略：減倉觀望、選擇權賣方、小部位試探
- 最佳指標：ATR、VIX、布林通道帶寬

## 體制識別的核心方法

### 方法一：ADX 趨勢強度判斷
- ADX > 25：趨勢體制（看 +DI/-DI 判斷多空方向）
- ADX 20-25：過渡區（趨勢可能形成或結束）
- ADX < 20：震盪體制
- **台股實證**：ADX 從 20 以下突破 25 是趨勢啟動訊號，從 25 以上跌破 20 是趨勢結束訊號

### 方法二：均線排列與扣抵
- 短中長期均線多頭排列 → 上升趨勢
- 短中長期均線空頭排列 → 下降趨勢
- 均線糾結 → 震盪或趨勢即將啟動
- **搭配扣抵值**：判斷均線即將上揚還是下彎，提前預判體制轉換

### 方法三：布林通道帶寬
- 帶寬擴張 + 價格貼近上軌 → 趨勢上升
- 帶寬擴張 + 價格貼近下軌 → 趨勢下降
- 帶寬收縮（Squeeze）→ 即將變盤，方向未定
- **Squeeze 後的突破方向決定新體制**

### 方法四：價格行為判斷
- 連續更高的高點與低點 → 上升趨勢
- 連續更低的高點與低點 → 下降趨勢
- 高低點大致持平 → 震盪
- **道氏理論**：主要趨勢、次級趨勢、微波三層次判斷

### 方法五：波動率指標
- VIX > 30：高波動體制，趋势可能剧烈但不可持续
- VIX 15-30：正常波动
- VIX < 15：低波動體制，適合賣方策略
- **台股對應**：台指選擇權隱含波動率

## 體制轉換的早期訊號

### 從趨勢轉為震盪
- ADX 從高位回落跌破 25
- 均線開始糾結
- 價格突破失敗頻率增加
- 成交量萎縮

### 從震盪轉為趨勢
- 布林通道 Squeeze 後帶寬急速擴張
- ADX 從低位突破 25
- 均線開始發散排列
- 成交量突然放大（突破量）

### 從低波動轉為高波動
- ATR 突然放大 2 倍以上
- VIX/隱含波動率跳升
- 連續跳空缺口
- 歷史波動率與隱含波動率同時上升

## 實戰操作框架：體制優先策略

### Step 1：每日體制判斷
1. 大盤 ADX 讀數確認趨勢或震盪
2. 均線排列確認多空方向
3. 布林帶寬確認波動程度
4. VIX/台指波動率確認風險等級

### Step 2：選擇匹配策略
- 體制明確 → 用對應策略（趨勢用順勢、震盪用區間）
- 體制過渡 → 減倉或觀望，不強做
- 體制不明 → 小部位試探，停損設緊

### Step 3：體制轉換時的應對
- 從趨勢轉震盪：順勢部位減碼，準備區間策略
- 從震盪轉趨勢：區間部位平倉，準備突破策略
- 體制不明時：現金為王，等待方向明朗

## 常見錯誤

1. **用震盪策略做趨勢行情**：均值回歸在趨勢中會不斷逆勢加碼，最終大虧
2. **用趨勢策略做震盪行情**：停損連發，手續費吃掉利潤
3. **忽略體制轉換訊號**：體制已經改變卻還在用舊策略
4. **強制交易不明的體制**：體制不明時最好的策略就是不交易
5. **只用單一指標判斷體制**：ADX 可能延遲，需要多指標共振確認
