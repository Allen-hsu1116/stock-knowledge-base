## 來源一：StockFeel 技術指標共振訊號

在技術分析的領域當中，很多的投資人喜歡透過技術指標來找到關鍵的買賣訊號。單純依靠技術指標到底能不能夠獲利？

### 單一技術指標的兩大問題

1. **太過頻繁的交易訊號**：例如使用KD指標進行買賣，黃金交叉買進、死亡交叉賣出，產生非常多的訊號，頻繁交易將導致龐大的手續費支出，過度侵蝕獲利，導致出現虧損。

2. **單一技術指標遇到錯誤訊號的機率過高**：單一技術指標提供的買賣訊號較為不可靠，因為還有很多因素會影響股價波動，若要提升交易的穩定性與準確度，就必須搭配其他指標一起使用。

### 什麼是指標共振？

所謂的指標共振指的是兩個以上的技術指標都發出同一個方向的訊號，代表有兩個指標支持這個方向的看法。

例如：
- 移動平均線（趨勢指標）上彎 + KD指標（擺盪指標）黃金交叉 → 趨勢多方控盤+多方轉折 → 買進勝率顯著提高
- MACD（趨勢指標）柱狀體為負 + RSI指標（擺盪指標）轉空 → 趨勢空方格局+空方轉折 → 很好的放空位置

### 指標共振就像投票機制

當MACD、MA、CCI指標都顯示這檔股票為多方格局，代表從各種分析面相來看都具備多頭跡象，多個指標都認同這個看法，後續真的持續上漲的機率就會比較高。

### 指標共振的實際範例

**前提**：必須使用兩種不同類型的技術指標！

如果同時只用兩款擺盪指標（如KD指標與RSI指標），因為同質性太高，出現同樣訊號是非常常見的狀況，無法提高勝率。

**案例一：MACD + KD（趨勢指標 + 擺盪指標）**
國巨（2327）2019年9月：MACD柱狀體在零軸之上（趨勢多方控盤），KD指標出現黃金交叉（擺盪指標多方轉折），股價迎來一波顯著飆漲。

**案例二：MA + KD（趨勢指標 + 擺盪指標）**
國巨（2327）2020年初：60MA呈現上彎（波段趨勢上漲格局），KD指標出現黃金交叉時股價都迎來上漲。

**案例三：Bollinger Band + RSI（趨勢指標 + 擺盪指標）**
台積電（2330）2016年7月：股價跌破布林通道下緣（超賣嫌疑），中線依然維持上彎（趨勢上漲），RSI來到40軸附近（偏向超賣水準），判斷為趨勢上漲格局下的佈局買點。

### 結論

使用單一技術指標容易遇到指標盲點，建議至少要把指標共振的技巧搭配使用，才能有效提升交易勝率與報酬率。在選擇搭配的技術指標時，必須選擇不同性質的技術指標，才能達到指標共振的優勢。在指標共振下，可以在對的趨勢下找到對的進場位置。

## 來源二：OCPay 技術指標共振策略運用指南

### 指標共振原理

指標共振指的是多於一個指標在特定時刻給出一致的趨勢信號。這代表了多個指標支持某一方向趨勢，提高了成功的概率。

### 實際運用案例

- 案例一：MACD + KD指標（趨勢 + 擺盪），經常帶來成功的交易決策
- 案例二：移動平均線（MA）+ KD，在上漲趨勢中選擇合適的進場時間點

### 注意事項

避免使用同類型的指標，以免誤判。應選擇綜合指標，結合不同市場趨勢及動機，才能充分利用共振的優勢。

## 來源三：鏈股調查局 - 技術分析交易策略全解析

### 核心理念：三大假設

1. 市場行為包容一切資訊
2. 價格會沿著趨勢移動
3. 歷史會重演

### 建立交易策略四步驟

1. **明確交易風格與目標**：短線/波段/長線
2. **選擇核心技術指標組合**：建議2-4個不同類型，黃金組合 = 1趨勢 + 1動能 + 成交量
3. **制定進出場規則SOP**：進場條件、停損條件、停利條件
4. **資金與風險管理**：單筆虧損1-2%、風險報酬比至少1:2

### 實戰案例

**MA + RSI 策略捕捉波段趨勢**
- 進場SOP：股價站穩60MA之上（確認多頭趨勢）+ RSI從30以下回升黃金交叉向上（超賣反轉）
- 出場SOP：停損跌破支撐或60MA；停利RSI進入80以上超買區+趨勢走緩或反轉K線

**布林通道 + 成交量進行短線交易**
- 進場SOP：布林通道收窄進入擠壓狀態 + 帶量突破上軌或下軌
- 出場SOP：停損設在布林通道中軌；停利當價格觸及相反軌道或動能減弱
