## 什麼是Beta係數

Beta（β）是衡量個股或投資組合相對於大盤波動程度的指標。它量化了系統性風險——即無法透過分散投資消除的市場風險。

公式：β = Cov(Ri, Rm) / Var(Rm)

- Ri = 個股報酬率
- Rm = 市場報酬率
- Cov = 共變異數
- Var = 變異數

## Beta值的解讀

| Beta值 | 意義 | 實例 |
|--------|------|------|
| β = 1 | 與大盤同步 | 0050、大型權值股 |
| β > 1 | 比大盤更波動 | 半導體股、成長股、小型股 |
| β < 1 | 比大盤穩定 | 公用事業、電信股、高股息股 |
| β < 0 | 與大盤反向 | 黃金、反向ETF |
| β = 0 | 與大盤無相關 | 現金、避險資產 |

## Beta的計算細節

### 迴歸模型

Beta來自於CAPM（資本資產定價模型）的迴歸式：
Ri - Rf = α + β(Rm - Rf) + ε

- α = 超額報酬（選股能力）
- β = 系統性風險暴露
- ε = 殘差（非系統性風險）

### 計算期間選擇

| 期間 | 適用場景 | 注意 |
|------|---------|------|
| 1年（250日） | 短期Beta，反映近期特徵 | 波動大，不穩定 |
| 2年（500日） | 平衡短期與穩定性 | 較常用 |
| 5年（1250日） | 長期結構性Beta | 最穩定，但可能落後 |

### 週期選擇

- 日報酬：噪音多，但樣本量大
- 週報酬：平衡噪音與樣本量
- 月報酬：最乾淨，但樣本量小

## Beta的實戰應用

### 1. 選股時的Beta考量

牛市策略：選β>1的股票，放大市場上漲報酬
熊市策略：選β<1或β<0的標的，降低下檔風險
震盪市：β≈0的標的，不隨大盤起舞

### 2. 投資組合的整體Beta

組合Beta = Σ（個股Beta × 個股權重）

範例：
- 50% 0050（β=1）+ 30% 中華電（β=0.5）+ 20% 期貨空單（β=-1）
- 組合β = 0.5×1 + 0.3×0.5 + 0.2×(-1) = 0.45
- 大盤漲10%，組合約漲4.5%

### 3. 調整投資組合Beta

想要降低組合風險：
- 方法一：賣出高Beta股票，買入低Beta股票
- 方法二：加入反向ETF或放空期貨
- 方法三：增加現金部位（β=0）

想要提高組合報酬：
- 方法一：增加高Beta成長股
- 方法二：使用槓桿ETF
- 方法三：融資買入（風險最高）

### 4. Beta與停損設定

高Beta股票需要更大的停損空間：
- β=1.5的股票，大盤跌2%時可能跌3%
- 停損要考慮個股波動性（ATR），不能只看百分比

### 5. 產業Beta特性（台股市場）

| 產業 | 典型Beta範圍 | 特性 |
|------|-------------|------|
| 半導體 | 1.2-1.8 | 高度景氣循環 |
| 金融 | 0.8-1.2 | 與利率週期相關 |
| 電信 | 0.4-0.7 | 防禦型，穩定現金流 |
| 營建 | 1.3-1.8 | 高Beta週期股 |
| 食品 | 0.5-0.8 | 消費必需品，低Beta |
| 生技 | 1.5-2.5 | 高波動，消息面驅動 |

## 注意事項

### Beta的五大陷阱

1. **後視鏡陷阱**：Beta是用歷史數據算的，過去的Beta不代表未來的Beta。公司業務轉型、併購後Beta可能完全改變

2. **不同市場不同Beta**：同一檔股票在多頭和空頭市場的Beta可能不同。下跌時Beta通常升高（不對稱Beta）

3. **R²太低無意義**：如果R²<0.3，表示個股走勢跟大盤幾乎無關，此時Beta沒有預測力。小型股、冷門股常見

4. **Beta不衡量非系統性風險**：一家公司可能Beta很低但非系統性風險極高（如公司治理問題、單一產品風險）

5. **槓桿影響Beta**：有負債的公司權益Beta比資產Beta高。財務槓桿越高，Beta越高

### 不對稱Beta（Downside Beta）

下行Beta > 上行Beta 是常見現象：
- 牛市時β=0.8的股票，熊市時β可能變成1.3
- 這意味「穩定」的股票在崩盤時可能比大盤跌更多
- 解決方案：分別計算上漲日和下跌日的Beta

### 估值中的Beta

CAPM公式：預期報酬 = Rf + β(Rm - Rf)

- Rf = 無風險利率
- Rm - Rf = 市場風險溢酬
- β越高，要求的報酬率越高
- DCF估值中，Beta直接影響折現率和估值
