## 用「外資成本線」判斷個股多空趨勢

「外資成本線」跟一般的均線(MA)一樣，能幫助我們判斷一檔股票的多空趨勢。然而，兩者最大的差別在於統計對象的不同：

- 一般的「均線」主要是統計全市場的成本
- 「外資成本線」則是統計外資的成本

值得注意的是，外資通常都是「波段操作」：

- 當一檔股票**站上外資成本線** → 代表外資幾乎都賺錢了，會繼續加碼，容易推升股價繼續上漲
- 當股價**跌破外資成本線** → 代表外資幾乎都套牢虧錢了，後續推升股價上漲的力道將減弱，外資停損快狠準，破了外資成本線往往造成連鎖效應，形成更大的賣壓

## 外資持股比例超過10% → 外資成本線更具參考性

觀察「外資成本線」之前，必須先確認該股票是不是「外資」所認養的！當「外資持股比率」超過10%以上，就可以視其為「外資」認養的股票。

## 外資成本線實戰用法

1. **股價在外資成本線之上 → 拉回是買點**
2. **股價在外資成本線之下 → 反彈是空點**

### 趨勢由空翻多（突破外資成本線）

股價第一次從外資成本線之下突破，先觀察3~5天，看看是否站穩外資成本線之上，確認不是假突破後，再進場買進。

建議投入1/3資金試單，因為趨勢由空翻多時，股價會先經過一陣多空交戰，最後才真正站穩外資成本線。如果一下子就買滿，假突破停損會損失不少手續費。

### 趨勢由多翻空（跌破外資成本線）

股價第一次從外資成本線之上跌破，先觀察3~5天，看看是否仍位於外資成本線之下，確認不是假跌破後，再進場放空。

### 第二次突破是更好的買點

有時候股價突破外資成本線後會出現解套賣壓，經過幾天震盪才往上。因此第二次突破外資成本線是比較好的買點，較能確認股價站穩。

## 3步驟挑出突破「外資成本線」清單

1. 勾選「突破外資成本線」
2. 設定「持股比例」大於10%
3. 挑出近5日買超最多的個股優先觀察

## FinLab：外資買入成本指標選股

### 計算方式

外資買進成本 = Σ(外資買進張數 × 收盤價) / Σ(外資買進張數)

使用60日rolling window計算：
```python
外資買進成本 = (外資買進 * 收盤價).rolling(60, min_periods=1).sum() / 外資買進.rolling(60, min_periods=1).sum()
外資賣出成本 = (外資賣出 * 收盤價).rolling(60, min_periods=1).sum() / 外資賣出.rolling(60, min_periods=1).sum()
```

### 選股邏輯

1. 當買進成本高於賣出成本時 → 可以考慮買進的時機
2. 當股價上漲後，拉回到買進成本線 → 也是很好的買點
3. 股價小於外資成本的股票清單 → 觀察是否有突破外資成本，一飛沖天
