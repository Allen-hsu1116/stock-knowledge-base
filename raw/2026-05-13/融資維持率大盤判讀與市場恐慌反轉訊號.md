# 融資維持率大盤判讀與市場恐慌反轉訊號

**日期**：2026-05-13
**主題**：籌碼面分析

## 原文內容

### 融資維持率的定義

融資維持率 = 名目股價 / (每股融資成本 × 融資乘數) × 100%

- 上市公司融資乘數：60%
- 上櫃公司融資乘數：50%

例子：100元買進上市股票
- 原融資維持率 = 100 / (100 × 60%) = 166%
- 股價跌至78元：維持率 = 78 / 60 = 130%（追繳線）
- 股價跌至60元：維持率 = 60 / 60 = 100%（高危狀態）

### 市場普遍標準

- 130%~160%：相對安全
- 低於130%：券商發出追繳通知，T+2補繳
- 低於100%：融資戶嚴重浮虧，斷頭風險極高

### 極低維持率出現原因

1. 高檔融資進場：散戶追高
2. 股價急跌：AI、航運、重電修正
3. 未及時補繳保證金
4. 市場恐慌引發連鎖斷頭

### 2025年4月實戰案例

台股單日下跌2,065點（跌幅9.6%），融資維持率降至133.8%，低於130%的個股達589檔。

個股案例：
- 緯創(3231)：AI題材大漲後融資買盤湧入，2025 Q1回檔，維持率下探至91%
- 技嘉(2376)：維持率跌破100%，引發多日追繳賣壓

### TEJ回測策略

**進場條件（三重濾網）**：
1. TEJ融資維持率 < 過去10日移動平均（異常低檔）
2. 當日成交量 < 過去10日平均量（籌碼穩定）
3. 當日紅K（收盤 > 開盤）（反轉訊號）
4. 當日融資餘額 > 前5日平均 × 0.95（融資戶仍撐著）

**操作規則**：
- 符合條件後紅K當日開盤價買進
- 持有15個交易日或停利+40%/停損-10%
- 15日內有新訊號以最新訊號重新判斷

**回測結果（2020-2025）**：
- 年化報酬率：18.39%（大盤19.04%）
- 年化波動度：17.67%（大盤16.897%）
- 夏普值：1.04（大盤1.12）
- 最大回撤：-28.47%（大盤-16.68%）

策略特點：最大回撤控制優於大盤，低波動環境下提供穩定報酬，融資維持率異常低檔是恐慌反轉的有效訊號。
