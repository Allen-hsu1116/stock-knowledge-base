## 融資維持率計算

融資維持率 = 名目股價 / (每股融資成本 × 融資乘數) × 100%

- 上市公司融資乘數：60%
- 上櫃公司融資乘數：50%

### 計算範例

以100元買進上市股票一張（1,000股），融資乘數60%：
- 原始融資維持率 = 100 / (100 × 60%) = 166%
- 股價跌至78元時 = 78 / 60 = 130%（觸發追繳）
- 股價跌至60元時 = 60 / 60 = 100%（高風險斷頭）

## 為何出現極低維持率？

1. **高檔融資進場**：散戶常在股價創高時追高，平均融資成本偏高
2. **股價急跌**：如AI、航運、重電類股修正，帳戶價值迅速縮水
3. **未及時補繳保證金**：投資人選擇觀望或無力補繳
4. **市場恐慌**：引發連鎖斷頭效應，融資餘額與維持率惡化

## 時事案例：2025年4月台股重挫

- 單日下跌2,065點，跌幅9.6%，台股史上第三大單日跌點
- 融資維持率降至133.8%
- 低於130%的個股達589檔
- 形成連環追繳潮，券商大量發出追繳通知
- 融資餘額短期未快速下降，投資人抱股觀望，拉長修正時間

### 個股案例

- **緯創(3231)**：2025年Q1融資維持率下探至91%，絕大多數融資戶浮虧
- **技嘉(2376)**：維持率跌破100%，引發多日追繳賣壓

## TEJ回測策略：融資維持率異常低檔反轉

### 進場條件（三重濾網）

1. **融資維持率異常低檔**：當日TEJ融資維持率 < 過去10日移動平均值
2. **成交量確認**：當日成交量 < 過去10日平均量（籌碼面趨於穩定，融資散戶已出場）
3. **紅K反轉訊號**：當日收盤價 > 開盤價（出現反轉時機）
4. **融資餘額穩定**：當日融資餘額 > 前5日平均融資餘額 × 0.95（剩餘融資戶仍撐著，反彈機率較高）

### 出場條件

- 持有15個交易日
- 停利 +40%
- 停損 -10%
- 若15日內再次觸發進場訊號，以最新訊號日重新計算出場條件

### 回測結果（2020-2025）

| 績效指標 | 融資維持率策略 | 大盤Benchmark |
|---------|-------------|-------------|
| 年化報酬率 | 18.39% | 19.04% |
| 累積報酬率 | 110.51% | 115.6% |
| 年化波動度 | 17.67% | 16.897% |
| 夏普值 | 1.04 | 1.12 |
| 卡瑪比率 | 0.65 | 1.14 |
| 期間最大回撤 | -28.47% | -16.68% |

### 策略評估

- 最大回撤控制優於大盤，有效降低極端波動衝擊
- 年化報酬率略高於大盤
- 累積報酬長期優於大盤
- 年化波動度低於大盤，風險調整後報酬佳
- 夏普值表現良好
