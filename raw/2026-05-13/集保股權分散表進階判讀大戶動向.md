# 集保股權分散表進階判讀大戶動向

**來源**：https://www.cmoney.tw/learn/course/0520/topic/1079 + https://www.tejwin.com/insight/認識集保股權分散/
**日期**：2026-05-13
**主題**：籌碼面分析

---

## 原文內容

### 集保股權分散表是什麼

臺灣集中保管結算所每週公告的股東持有股數變化表，記錄每檔股票不同持股級別的人數和佔集保庫存比例。

### 核心判讀邏輯

**大股東持股增加 → 看好未來**
- 千張以上大戶持股比例高 = 籌碼穩定
- 大戶最了解公司經營狀況，大量持有代表看好
- 籌碼穩定的股票容易被拉抬，股本越小效果越明顯

**大股東持股減少 → 風險存在**
- 大戶不看好，股票流落到散戶手中
- 散戶容易因漲跌拋售，籌碼散亂
- 籌碼散亂不易拉抬，利空時容易大跌

### 持股分級範例（台積電2330）

- 1-999股（不到1張）：57萬人，佔集保庫存0.35%
- 1,000,001股以上：極少數人，佔集保庫存88.20%

台積電大戶持股高度集中，散戶影響力極低。

### 大戶動向追蹤

透過主力籌碼趨勢圖追蹤大戶操作行為：
- 大戶持續增加 → 利多
- 大戶持續減少 → 利空

### 資料更新頻率

- 週資料：每週最後營業日公告（含前一週四的資料）
- 月資料：每月更新

### 實戰注意事項

1. 一個月才公布一次，屬於落後指標
2. 搭配日/週的法人買賣超和分點進出交叉驗證
3. 千張大戶增減是核心觀察指標
4. 小股東（1-5張）變化判斷股票是否過熱
