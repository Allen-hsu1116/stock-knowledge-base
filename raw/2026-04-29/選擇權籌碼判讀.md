## 永豐期貨：選擇權入門與未平倉解讀

### 選擇權基本概念

選擇權邏輯跟保險類似：
- 擔心上漲 → 買 Call（買權）
- 擔心下跌 → 買 Put（賣權）
- 也可以當賣方：賣 Call（看不漲）、賣 Put（看不跌）

### 選擇權風險

- **買方**：風險有限（權利金），但勝率較低，時間價值逐漸流逝
- **賣方**：勝率較高，但遇到大行情可能一次賠回數週獲利

### 四種基本策略

| 策略 | 使用時機 | 最大損失 | 最大獲利 |
|------|----------|----------|----------|
| Buy Call | 看大漲 | 權利金 | 無限 |
| Buy Put | 看大跌 | 權利金 | 履約價-權利金 |
| Sell Call | 看不漲 | 無限 | 權利金 |
| Sell Put | 看不跌 | 履約價-權利金 | 權利金 |

### 選擇權未平倉五大解讀面向

1. **OI的意義**：未平倉量越多，代表越多資金押注，OI = 市場籌碼數據
2. **以賣方角度解讀**：Call OI增加 → 看不漲（壓力）；Put OI增加 → 看不跌（支撐）
3. **Put/Call Ratio**：總Put OI / 總Call OI，>1偏多，<1偏空
4. **OI大量區**：Call最大OI區 = 壓力區；Put最大OI區 = �撐區
5. **三大法人選擇權未平倉**：外資、投信、自營商的Call/Put留倉判斷多空

## Winsmart：選擇權未平倉4個重點指標

### 1. 未平倉量是市場燃料

OI增加 = 多空分歧加大 = 趨勢燃料
OI減少 = 輸家認賠出場 = 趨勢可能反轉

### 2. 以賣方（莊家）視角解讀

賣方需要繳保證金、承擔較高風險，因此賣方OI更能反映大戶看法。
- Call OI大量區 = 賣方認為不會漲過 = 壓力
- Put OI大量區 = 賣方認為不會跌破 = 支撐

### 3. 成交量與OI的關係

| 指數 | OI | 成交量 | 判斷 |
|------|----|--------|------|
| 上漲 | 增加 | 增加 | 強勢軋空 |
| 上漲 | 增加 | 減少 | 多頭持續 |
| 上漲 | 減少 | 增加 | 多頭反轉 |
| 下跌 | 增加 | 增加 | 強勢走空 |
| 下跌 | 減少 | 增加 | 空頭反轉 |

### 4. Put/Call Ratio

- P/C Ratio > 100% → 偏多（Sell Put > Sell Call）
- P/C Ratio < 100% → 偏空
- 極端值：> 115% 多頭趨勢，< 75% 空頭趨勢
- 與大盤指數呈正向關係

### 5. VIX（恐慌指數）

- 多頭行情：緩漲，VIX緩步下降
- 空頭行情：急跌，VIX急速飆高
- 選擇權賣方偏好多頭（波動率下降有利）
- 大事件前VIX上升（選舉、財報等）

### 6. 支撐壓力判斷

從OI最大量找出支撐壓力區，訂定操作策略：
- 突破壓力區 → 「倒莊」效應，可能加速上漲
- 跌破支撐區 → 可能加速下跌

## Gooptions：法人選擇權籌碼分析

### 自營商角色：造市者

自營商是「造市者」（Market Maker），有人要交易就必須提供流動性：
- 外資買Call 1億 → 自營商必須賣Call 1億
- 自營商的部位是被動產生的，不是主動看好/看壞

### 自營商避險解讀

自營商賣出Call後，為了避免大漲時巨虧，會買期貨多單避險：
- 外資買Call + 自營商賣Call → 自營商買期貨避險
- 這就是為什麼自營商期貨部位可以解讀外資選擇權動向

### 三大法人選擇權判讀重點

1. **外資**：選擇權最重要的籌碼指標
   - 外資大買Call → 看多
   - 外資大買Put → 看空
   - 外資賣Call + 買Put → 強烈看空

2. **投信**：在選擇權市場部位較小，參考性有限

3. **自營商**：造市+避險，不是方向性指標
   - 自營商的Call/Put部位是造市結果
   - 要看的是自營商如何避險（期貨多空方向）

### 選擇權支撐壓力表

- Call最大OI的履約價 = 壓力區
- Put最大OI的履約價 = 支撐區
- 留意鄰近履約價的OI增減變化
- OI明顯增加的區域，支撐壓力效果更顯著
