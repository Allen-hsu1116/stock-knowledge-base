## 文章一：台股籌碼分析完整教學（Wistock）

### 什麼是籌碼分析？

籌碼分析是追蹤股票市場中「誰在買、誰在賣、買賣了多少」的分析方法。台股每日成交量中，法人交易佔比約 40%-60%，股價短期走勢很大程度被法人資金推動。

### 股市角色定位

- **大戶與主力**：外資法人、投信基金、自營商、資金雄厚的大戶。資金量大、資訊優勢明顯、操作有計畫性。外資在台股持股市值佔比長期維持 40%。
- **中實戶**：資金數百萬到數千萬。觀察主力動向搭順風車，撤退速度也快。
- **散戶**：資金小、資訊慢、容易被情緒左右。最常犯的錯：「買在主力出貨高點，賣在主力吸貨低點」。

### 法人三大類買賣超判讀

**外資**：
- 以大型權值股為主（台積電、聯發科、鴻海）
- 連續買超代表國際資金看好台股
- 「現貨買超 + 期貨多單增加」才是真正偏多訊號
- 單看現貨買賣超可能被誤導

**投信**：
- 偏好中小型成長股
- 看好時會連續買進數天甚至數週
- 季底（3/6/9/12月）有作帳行情

**自營商**：
- 短進短出，今天買明天賣很常見
- 參考價值相對低
- 三大法人同步買超才是強多方訊號

### 融資融券解讀

- **融資增加**：散戶看多。低檔溫和增加 OK，高檔暴增要小心（多殺多連鎖反應）
- **融資餘額下降 + 股價不跌**：散戶在賣，有人在接（很可能是主力）
- **融券增加**：市場看空，但可能引發軋空行情
- **券資比 > 30%**：軋空風險明顯升高

### 券商分點資料

- 可看到每檔股票在各券商分點的買賣明細
- 特定主力或大戶通常固定在幾個分點下單
- 常勝分點連續五天大量買超（>當日成交量3%）+ 股價未明顯拉升 = 典型主力吸貨訊號
- 主力會分散分點下單隱藏蹤跡，或用對敲製造假象

### 籌碼集中度判讀方式

- **千張大戶持股比例持續上升** → 籌碼往大戶集中
- **融資餘額下降但股價不跌** → 散戶賣、主力接
- **成交量萎縮到極低、股價穩定** → 籌碼沉澱，浮額清洗乾淨
- **券商分點連續大量買超** → 特定主力吸貨
- 兩個以上訊號同時出現 → 值得列入觀察名單

### 主力集中價與散戶解套價

- **主力集中價**：主力進場成本的平均價位。站上→主力獲利，有動機推升；跌破→主力可能停損或觀望
- **散戶解套價**：散戶平均持股成本估算值。接近時形成解套賣壓
- 關鍵：不是怕解套賣壓，而是看「解套賣壓釋放後股價有沒有守住」。守住代表有人在接（主力）

### 主力吸貨訊號實戰案例

「法人買、主力買、散戶賣、量增價穩」的組合 = 教科書級主力吸貨訊號：
- 投信連續買超
- 特定分點大量買超
- 融資餘額減少（散戶在賣）
- 帶量小漲（主力壓著吃貨）

### 散戶常犯五大籌碼判讀錯誤

1. **只看單日數據就下結論**：至少觀察3-5天累積買賣超
2. **忽略量價配合**：同樣的籌碼數據，不同量價結構解讀完全相反
3. **把籌碼分析當唯一依據**：要搭配基本面一起用
4. **忽略主力製造假象**：對敲洗盤、放利多引散戶進場再出貨
5. **不記錄觀察結果**：持續記錄籌碼訊號和後續走勢才能累積判斷力

## 文章二：主力連買13天，籌碼集中大漲74.1%（CMoney）

### 觀察主力的三大指標

**主力買賣超**：
- 數值為正 → 主力買進，連續數天看多
- 數值為負 → 主力賣出，連續數天看空

**買賣家數差**：
- 家數差 > 0 → 買家數 > 賣家數，籌碼分散，不易上漲
- 家數差 < 0 → 買家數 < 賣家數，籌碼集中，容易拉抬

**籌碼集中度**：
- 定義：一段時間內，買超前15名券商總張數 - 賣超前15名券商總張數 ÷ 成交量
- 短天期集中度 > 長天期集中度 → 近期主力看好持續買入
- 籌碼集中是股價上漲的前兆

### 操作策略

**最佳進場訊號**：
- 主力買賣超連續數日正數
- 買賣家數差連續數日負數
- 籌碼集中度上升
- 三指標共振 = 良好進場時機

**注意**：漲幅已大時不追高，容易吸引追價者導致籌碼分散

### 實戰案例：鴻海(2317)

- 股價從100.5元漲到175元，漲幅74.1%
- 主力連買近14天，力道強勁
- 買賣家數差多為綠柱（籌碼往少數人集中）
- 上漲前長短天期籌碼集中度都向上成長
- 籌碼集中 + 主力連買 = 飆股動能
