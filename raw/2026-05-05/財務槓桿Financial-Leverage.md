# 財務槓桿 Financial Leverage

**來源**：https://wiki.mbalib.com/zh-tw/%E8%B4%A2%E5%8A%A1%E6%9D%A0%E6%9D%86
**日期**：2026-05-05
**主題**：基本面分析

---

財務杠桿是指由於債務的存在而導致普通股每股利潤變動大於息稅前利潤變動的杠桿效應。

## 財務槓桿的定義

1. 將財務杠桿定義為「企業在制定資本結構決策時對債務籌資的利用」，又稱融資杠桿、資本杠桿或負債經營
2. 認為財務杠桿是指在籌資中適當舉債，調整資本結構給企業帶來額外收益。若負債經營使每股利潤上升→正財務杠桿；若使每股利潤下降→負財務杠桿

## 財務槓桿的作用

財務杠桿作用是負債和優先股籌資在提高企業所有者收益中所起的作用，以企業的投資利潤率與負債利息率的對比關係為基礎。

1. **投資利潤率 > 負債利息率**：正財務槓�，企業獲得額外收益（財務槓桿利益）
2. **投資利潤率 < 負債利息率**：負財務槓桿，企業承擔額外損失（財務風險）

## 財務槓桿係數（DFL）

財務杠桿繫數是指普通股每股利潤的變動率相當於息稅前利潤變動率的倍數。

計算公式：
- DFL = 普通股每股利潤變動率 / 息稅前利潤變動率
- 簡化公式：DFL = EBIT / (EBIT - I)

其中：
- EBIT = 息稅前利潤
- I = 債務利息

含優先股時：DFL = EBIT / [EBIT - I - 融資租賃租金 - 優先股股利/(1-所得稅稅率)]

## DFL的含義

- DFL越大，財務風險越大，但預期每股收益也越高
- DFL = 1：無負債，無財務槓桿效應
- 負債比率越高，DFL越高

### A、B、C三公司比較案例

| 項目 | A公司 | B公司 | C公司 |
|------|-------|-------|-------|
| 普通股本 | 2,000,000 | 1,500,000 | 1,000,000 |
| 債務（利率8%） | 0 | 500,000 | 1,000,000 |
| 資本總額 | 2,000,000 | 2,000,000 | 2,000,000 |
| EBIT | 200,000 | 200,000 | 200,000 |
| 債務利息 | 0 | 40,000 | 80,000 |
| DFL | 1 | 1.25 | 1.67 |
| 每股收益 | 6.7 | 7.15 | 8.04 |

EBIT增長1倍時：
- A公司每股收益增長1倍（13.4/6.7-1）
- B公司每股收益增長1.25倍（16.08/7.15-1）
- C公司每股收益增長1.67倍（21.44/8.04-1）

## 財務槓桿的影響因素

1. **息稅前利潤率**：在資本總額和資本結構相同下，EBIT率高低影響DFL
2. **負債利息率**：利息率高低影響DFL和每股收益
3. **資本結構**：負債比率是影響DFL的關鍵因素

## 財務槓桿與財務風險的關係

- 財務風險主要由財務槓桿產生
- 財務杠桿會加大財務風險，企業舉債比重越大，財務風險越大
- 負債比率是可以控制的，企業可通過合理安排資本結構，適度負債

## 資本結構和財務槓桿的關係

EPS = (EBIT - I)(1-T) / N

資本結構中增加負債或優先股籌資比例時，固定現金流出量增加，從而加大DFL和財務風險。

如果全部資產收益率低於固定籌資費率，普通股收益率就會低於公司投資收益率或出現資本虧損。

## 金融危機下的教訓

2008年貝爾斯登因次貸危機倒閉，總資產與股東權益之比高達16倍。較高的負債經營雖然能提高投資回報率，但必定也會伴隨較大的財務風險。
