# 股東權益報酬率（ROE）

**來源**：https://wiki.mbalib.com/zh-tw/股東權益報酬率
**日期**：2026-05-05
**主題**：基本面分析

---

## 定義

淨資產收益率又稱股東權益報酬率/淨值報酬率/權益報酬率/權益利潤率/淨資產利潤率，是衡量上市公司盈利能力的重要指標。是指利潤額與平均股東權益的比值，該指標越高，說明投資帶來的收益越高；淨資產收益率越低，說明企業所有者權益的獲利能力越弱。該指標體現了自有資本獲得淨收益的能力。

一般來說，負債增加會導致淨資產收益率的上升。

## 計算公式

- 淨資產收益率 = 淨利潤 / 平均淨資產 × 100%
- 其中，平均淨資產 = （年初淨資產 + 年末淨資產）/ 2
- 也可以使用年末淨資產：淨資產收益率 = 淨利潤 / 年度末股東權益 × 100%

## 杜邦公式

淨資產收益率（ROE）= 淨利潤 / 所有者權益 = 銷售淨利潤率 × 資產周轉率 × 權益乘數（財務槓桿）

- 銷售淨利潤率 = 淨利潤 / 銷售收入（盈利能力）
- 資產周轉率 = 銷售收入 / 總資產（營運能力）
- 權益乘數 = 資產總額 / 股東權益總額 = 1 / (1 - 資產負債率)（償債能力）

## 影響因素分析

1. **總資產報酬率**：淨資產是企業全部資產的一部分，因此淨資產收益率必然受企業總資產報酬率的影響。在負債利息率和資本構成等條件不變的情況下，總資產報酬率越高，淨資產收益率就越高。

2. **負債利息率**：在資本結構一定情況下，當總資產報酬率高於負債利息率時，將對淨資產收益率產生有利影響；反之，將產生不利影響。

3. **資本結構（負債與所有者權益之比）**：當總資產報酬率高於負債利息率時，提高負債比例將使淨資產收益率提高；反之則降低。

4. **所得稅率**：所得稅率提高，淨資產收益率下降；反之則上升。

## 因素分析公式

淨資產收益率 = [總資產報酬率 + (總資產報酬率 - 負債利息率) × 負債/淨資產] × (1 - 所得稅率)

## 實例分析

某公司1999年 vs 2000年資料：

| 項目 | 1999年 | 2000年 |
|------|--------|--------|
| 平均總資產 | 69,491 | 100,731 |
| 平均淨資產 | 47,090 | 79,833 |
| 負債 | 22,401 | 20,898 |
| 負債與淨資產之比 | 0.4757 | 0.2618 |
| 利息支出（平均利率） | 1,744 (7.79%) | 1,525 (7.30%) |
| 淨利潤（所得稅率） | 7,743 (21.34%) | 8,431 (16.23%) |
| 總資產報酬率 | 16.68% | 11.50% |
| 淨資產收益率 | 16.44% | 10.56% |

因素分析結果：
- 總資產報酬率變動的影響：-6.01%
- 利息率變動的影響：+0.19%
- 資本結構變動的影響：-0.71%
- 稅率變動的影響：+0.56%
- 合計：-5.88%

可見公司2000年淨資產收益率下降主要是由於總資產報酬率下降引起的。
