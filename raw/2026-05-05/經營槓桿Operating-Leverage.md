# 經營槓桿 Operating Leverage

**來源**：https://wiki.mbalib.com/zh-tw/%E8%90%A5%E4%B8%9A%E6%9D%A0%E6%9D%86
**日期**：2026-05-05
**主題**：基本面分析

---

經營杠桿又稱營業杠桿或營運杠桿，指在企業生產經營中由於存在固定成本而導致息稅前利潤變動率大於產銷量變動率的規律。

根據成本性態，在一定產銷量範圍內，產銷量的增加一般不會影響固定成本總額，但會使單位產品固定成本降低，從而提高單位產品利潤，並使利潤增長率大於產銷量增長率；反之，產銷量減少，會使單位產品固定成本升高，從而降低單位產品利潤，並使利潤下降率大於產銷量的下降率。

## 經營槓桿係數計算公式

經營杠桿繫數（Degree of Operating Leverage，DOL），也稱營業杠桿程度，是息稅前利潤的變動率相當於銷售額（營業額）變動率的倍數。

計算公式為：
- DOL = 息稅前利潤變動率 / 產銷業務量變動率
- 簡化公式：DOL = 基期邊際貢獻 / 基期息稅前利潤 = Q(P-V) / [Q(P-V)-F]

其中：
- Q = 銷售數量
- P = 銷售單價
- V = 單位變動成本
- F = 固定成本總額

## 營業槓桿利益與營業風險

### 營業槓桿利益
在擴大銷售額的條件下，由於經營成本中固定成本相對降低，所帶來增長程度更快的經營利潤。

例：XYZ公司固定成本800萬，變動成本率60%
- 1998年營業額2400萬，利潤160萬
- 1999年營業額2600萬（+8%），利潤240萬（+50%）
- 2000年營業額3000萬（+15%），利潤400萬（+67%）

### 營業風險
營業風險也稱經營風險，是指與企業經營相關的風險，尤其是指利用營業杠桿而導致息稅前利潤變動的風險。

反向案例：
- 1998年營業額3000萬，利潤400萬
- 1999年營業額2600萬（-13%），利潤240萬（-40%）
- 2000年營業額2400萬（-8%），利潤160萬（-33%）

## DOL的一般規律

1. DOL體現了利潤變動和銷量變動之間的變化關係
2. DOL越大，經營風險越大
3. 固定成本不變，銷售額越大，DOL越小，經營風險越小
4. 當銷售額達到盈虧臨界點時，DOL趨近於無窮大
5. 固定成本是產生經營槓桿效應的根源，α=0時DOL=1，無經營槓桿效應

## 影響經營槓桿的因素

- 銷售數量↑ → DOL↓（反方向）
- 銷售單價↑ → DOL↓（反方向）
- 單位變動成本↑ → DOL↑（同方向）
- 固定成本↑ → DOL↑（同方向）

## 控制經營槓桿的途徑

1. 增加銷售金額
2. 降低產品單位變動成本
3. 降低固定成本比重

## 經營槓桿在企業不同生命週期的應用

1. **引入期**：經營風險高，應採權益籌資方式，降低還款付息壓力
2. **發展期**：資金需求高，需將DOL控制在一定範圍內
3. **成熟期**：現金流量大，可充分利用經營槓桿，採取高效投資策略
4. **衰退期**：市場占有率低，應轉移生產能力，採取新產品發展策略

## 經營槓桿、財務槓桿、聯合槓桿的關係

聯合杠桿是經營杠桿、財務杠桿的綜合反映，直接反映經營杠桿、財務杠桿對企業的共同影響。兩個杠桿是相輔相成的，存在互逆關係，故籌資和投資時應相互配合使用。
