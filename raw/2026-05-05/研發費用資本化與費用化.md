# 研發費用資本化與費用化

**來源**：https://wiki.mbalib.com/zh-tw/研发费用
**日期**：2026-05-05
**主題**：基本面分析

---

研發費用（R&D expenses）是指研究與開發某項目所支付的費用。

## 三種會計處理方式

1. **全部費用化**：所有研發支出在發生時計入當期損益
2. **全部資本化**：所有研發支出作為無形資產，在受益期內分攤
3. **有條件資本化**（現行準則）：研究階段費用化，開發階段符合條件時資本化

## 現行準則：研究 vs 開發

### 研究階段 → 費用化
探索性、不確定性高，支出於發生時計入當期損益（管理費用）。

### 開發階段 → 符合條件資本化
必須同時滿足五個條件：
1. 技術上完成該無形資產具有可行性
2. 具有完成並使用或出售的意圖
3. 能證明產生未來經濟利益（存在市場或有內部使用價值）
4. 有足夠的技術、財務資源支持完成
5. 支出能夠可靠計量

無法區分研究與開發階段 → 全部費用化。

## 研發費用構成

- 薪金、工資及其他人事費用
- 材料成本和已消耗的勞務
- 設備與設施的折舊
- 製造費用的合理分配
- 其他成本

## 稅法優惠

- 未形成無形資產的：在據實扣除基礎上，加計扣除50%（即扣除150%）
- 形成無形資產的：按無形資產成本的150%攤銷

## 有條件資本化的優點

1. 符合真實性原則：資產負債表反映研發投資力度
2. 體現謹慎性原則：開發階段才資本化，研究風險已排除
3. 避免短期行為：不會因費用化壓低利潤而減少研發投入
4. 遵循配比原則：未來收益期間分期攤銷

## 有條件資本化的缺陷

1. **操作難度大**：研究與開發階段劃分有主觀性，可能被用來操縱盈餘
2. **不符合一貫性原則**：跨年度研發，若僅當年資本化，資產價值反映不全面
3. **未體現重要性原則**：小額研發也資本化，增加核算工作量

## 案例計算

甲公司研發支出共1600萬元：
- 研究階段：320萬 → 費用化
- 開發階段（資本化前）：580萬 → 費用化
- 開發階段（資本化後）：700萬 → 資本化為無形資產

費用化合計 = 320 + 580 = 900萬
資本化 = 700萬，按10年直線法攤銷，每年攤銷70萬

稅法加計扣除：
- 費用化部分：1600萬 × 150% = 2400萬扣除額，但費用化僅900萬已計入損益
- 資本化部分：按700萬 × 150% = 1050萬攤銷
