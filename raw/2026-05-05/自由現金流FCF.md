# 自由現金流（Free Cash Flow, FCF）

## 定義

自由現金流（Free Cash Flow, FCF）是一種財務方法，用來衡量企業實際持有的能夠回報股東的現金。指在不危及公司生存與發展的前提下可供分配給股東（和債權人）的最大現金額。

簡單地講，自由現金流量就是企業產生的在滿足了再投資需要之後剩餘的現金流量。如果自由現金流豐富，則公司可以償還債務、開發新產品、回購股票、增加股息支付。

## 歷史發展

自由現金流最早由美國西北大學拉巴波特（Alfred Rappaport）、哈佛大學詹森（Michael Jensen）等學者於20世紀80年代提出。安然后、世通等績優公司紛紛破產後，FCF已成為企業價值評估領域使用最廣泛的指標，美國證監會更要求公司年報中必須披露此指標。

### 主要學者定義演進

1. **拉巴波特（Rappaport, 1986）**：確立五個價值驅動因素——銷售增長率、邊際營業利潤、新增固定資產投資、新增營運資本、資本成本。

2. **詹森（Jensen, 1986）**：提出自由現金流量理論（Agency Cost），定義FCF為滿足所有NPV為正的項目後剩餘的現金流量，認為這些資金應回報給股東。

3. **科普蘭（Copeland, 1990）**：FCF = (稅後淨營業利潤 + 折舊及攤銷) - (資本支出 + 營運資本增加)

4. **達姆達蘭（Damodaran, 1996）**：FCF = EBIT × (1 - 稅率) + 折舊 - 資本支出 - 營運資本增加

## FCF的分類

### 公司自由現金流量（FCFF）
公司支付所有營運費用、進行必需的固定資產與營運資產投資後，可以向所有投資者分派的稅後現金流量。

FCFF = EBIT × (1 - 稅率) + 折舊 - 資本支出 - 追加營運資本

### 股權自由現金流量（FCFE）
扣除所有開支、稅收支付、投資需要以及還本付息支出之後的剩餘現金流量。

FCFE = 淨收益 + 折舊 - 資本性支出 - 營運資本追加額 - 債務本金償還 + 新發行債務

### 關係
自由現金流量 = 股權自由現金流量 + 債權自由現金流量

## FCF的優點（相比會計利潤）

1. **排除非營業活動操縱**：只有持續核心業務產生的營業利潤才是可持續發展的源泉
2. **收付實現制**：會計利潤按權責發生制，FCF按收付實現制
3. **考慮存貨影響**：6天和60天的存貨對利潤沒影響，但對FCF有影響
4. **考慮股權資本成本**：FCF是扣除所有資本成本後的剩餘

安然后破產前二個季度季報還顯示盈利4.23億美元，可現金流量為負5.27億美元——這就是只看利潤的風險。

## FCF的局限性

1. 預測FCF需要大量假設
2. 不同學者對FCF的定義不盡相同
3. 適用於穩定增長的企業，新創或快速擴張期公司FCF可能為負
4. 需要配合其他指標一起判斷
