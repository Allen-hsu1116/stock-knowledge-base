# 無形資產攤銷與減值準備

**來源**：https://wiki.mbalib.com/zh-tw/无形资产摊销
**日期**：2026-05-05
**主題**：基本面分析

---

無形資產攤銷是指將使用壽命有限的無形資產的應攤銷金額在其使用壽命內進行系統合理地分配。包括專利權、商標權、著作權、土地使用權、非專利技術等無形資產的攤銷。

## 攤銷基本規則

1. 攤銷開始月份為取得當月
2. 攤銷方法為直線法（無法可靠確定預期實現方式時）
3. 攤銷年限一般為預計使用年限
4. 殘值應假定為零

## 攤銷年限確定原則

1. 合同規定受益年限但法律沒有規定有效年限 → 不超過合同受益年限
2. 合同沒有規定但法律規定有效年限 → 不超過法律有效年限
3. 兩者都有規定 → 不超過較短者
4. 兩者都沒規定 → 不超過10年

## 攤銷方法

- **平均攤銷法**：直線法、生產總量法、車流量法
- **加速攤銷法**：雙倍餘額遞減法、年數總和法

## 無形資產減值準備與累計攤銷的區別

| 維度 | 累計攤銷 | 減值準備 |
|------|---------|---------|
| 性質目的 | 價值分攤，不表示實際減少 | 確保現時價值計量有用相關 |
| 理論基礎 | 權責發生制 | 謹慎性原則 |
| 核算原則 | 配比原則 | 謹慎性原則 |
| 核算範圍 | 使用壽命有限的無形資產 | 所有無形資產（壽命不確定的每年必須減值測試）|
| 計提時間 | 按月進行 | 資產負債表日或年末 |
| 計提方法 | 直線法/加速法等 | 可收回金額 vs 帳面價值 |
| 計提基數 | 原值 - 預計殘值 - 已提減值準備 | 帳面價值 - 可收回金額 |
| 賬務處理 | 借：管理費用 / 貸：累計攤銷 | 借：資產減值損失 / 貸：無形資產減值準備 |

## 會計分錄

攤銷時：
- 借：管理費用——無形資產攤銷
- 貸：累計攤銷

減值時：
- 借：資產減值損失
- 貸：無形資產減值準備
