# 成交量特殊型態判讀：窒息量、凹洞量、遞增量與U型量

**來源**：知識庫整合（成交量確認原則、量價8大口訣、高檔爆量與底部爆量、量先價行、OBV等既有文章交叉整理）

**主題**：技術分析 - 成交量特殊型態判讀

---

## 一、成交量特殊型態總覽

成交量不只是「量大量小」的二分法，不同型態的成交量蘊含不同的市場訊息。特殊型態是主力行為和市場心理的指紋，看懂型態就能判斷下一步方向。

**六大特殊型態**：
1. 窒息量（地量）
2. 凹洞量（量坑）
3. 遞增量（量潮）
4. U型量（量谷）
5. 脈衝量（單日爆量）
6. 量滾量（換手量）

---

## 二、窒息量（地量）

### 定義
成交量萎縮到近期最低水平，通常低於20日均量的50%以下，甚至出現近3-6個月的新低量。

### 判斷標準
- 成交量 < 20日均量的50%
- 連續3日以上低量
- 換手率極低（大盤<0.3%，個股<0.5%）

### 位置意義

| 位置 | 意義 | 操作 |
|------|------|------|
| 長期底部 | 最後洗盤，浮額清洗完畢 | 觀察帶量突破進場 |
| 下跌中繼 | 只是暫時休息，下跌未止 | 續抱空單或觀望 |
| 盤整末期 | 方向即將選擇 | 等待帶量方向確認 |
| 上漲中繼 | 換手完成，籌碼穩定 | 多頭續抱 |

### 實戰要點
1. 窒息量不是進場訊號，是「準備進場」的訊號
2. 窒息量之後的帶量長紅才是真正買點
3. 窒息量持續天數越多，後續爆發力越大
4. 大盤窒息量比個股窒息量更可靠（個股可能因為冷門而常態性低量）

### 窒息量的陷阱
- 冷門股常態性低量不等於窒息量
- 停板鎖住的低量不是窒息量
- 除息前後的量縮不代表窒息量

## 三、凹洞量（量坑）

### 定義
成交量從正常水平突然萎縮到極低，再逐漸恢復，形成一個「凹洞」形狀。與窒息量的區別在於持續時間較短（通常3-5日），且前後有明確的量能基準線。

### 識別方法
1. 前後成交量維持在均量水平
2. 中間出現明顯的量能凹陷
3. 凹洞期間成交量低於5日均量
4. 凹洞後量能快速恢復

### 型態分類

| 型態 | 特徵 | 意義 |
|------|------|------|
| 底部凹洞 | 長期下跌後出現 | 主力悄悄吸籌，浮額洗盡 |
| 途中凹洞 | 上升趨勢中回測 | 換手完成，攻擊再起 |
| 頭部凹洞 | 高檔盤整中出現 | 主力不願追高，動能衰退 |

### 實戰操作
1. **凹洞量底部進場法**：凹洞量結束後第一根帶量長紅進場
2. **凹洞量確認法**：凹洞期間K線實體縮小，低點未破前低
3. **凹洞量陷阱**：凹洞後繼續量縮下跌，說明不是換手而是出貨

## 四、遞增量（量潮）

### 定義
成交量隨價格上漲逐日放大，形成「量潮」。每一根K線的成交量都大於前一日，是多方攻擊最健康的量能型態。

### 判斷標準
- 連續3日以上成交量遞增
- 價格同步上漲（量增價漲）
- 遞增過程中單日量不小於5日均量

### 三種遞增量型態

| 型態 | 特徵 | 力道 |
|------|------|------|
| 溫和遞增 | 每日增量5-10% | 穩健多頭，續航力最強 |
| 加速遞增 | 每日增量20-30% | 攻擊波，注意後續量能是否可持續 |
| 爆量遞增 | 每日增量50%以上 | 短線噴出，可能形成頭部 |

### 遞增量與均量線的關係
1. 5日均量上穿20日均量 → 短線攻擊啟動
2. 5日均量上穿60日均量 → 中線趨勢確立
3. 三條均量線多頭排列 → 長多格局

### 遞增量的衰減訊號
- 遞增幅度開始縮小（第1天增50%→第3天只增5%）
- 價格上漲但量能開始遞減（量價背離）
- 單日爆量後隔日量能急縮（換手失敗）

## 五、U型量（量谷）

### 定義
成交量從高水平逐漸萎縮到谷底，再逐漸放大回高水平，形成U型。與凹洞量的區別在於U型量的萎縮和恢復過程更緩慢、更對稱。

### 識別方法
1. 左側量能遞減（約5-10日）
2. 底部量能穩定在低水平（約3-7日）
3. 右側量能遞增（約5-10日）
4. 左右大致對稱

### 位置意義

| 位置 | 意義 | 確認條件 |
|------|------|----------|
| 底部U型 | 主力緩慢吸籌 | 右側放量突破頸線 |
| 上升途中U型 | 中繼換手 | 右側量能超過左側 |
| 頭部U型 | 主力緩慢出貨 | 右側價格無法創新高 |

### 與型態學的對應
- 底部U型量 + W底 = 高勝率買點
- 頭部U型量 + M頭 = 高勝率賣點
- 途中U型量 + 旗形 = 順勢加碼點

## 六、脈衝量（單日爆量）

### 定義
單日成交量突然暴增到平日的3倍以上，隔日迅速恢復正常水平。像心電圖上的脈衝，一閃而過。

### 位置判斷

| 位置 | 意義 | 後續走勢 |
|------|------|----------|
| 長期底部 | 主力搶進，利多消息刺激 | 觀察後續量能是否持續 |
| 上升初期 | 突破起漲，換手積極 | 利多，可追價 |
| 上升末期 | 主力對倒出貨 | 利空，準備出場 |
| 下跌途中 | 恐慌性拋售 | 可能是底部訊號 |

### 實戰要點
1. 脈衝量隔日量能回歸是正常現象
2. 關鍵是脈衝量之後的3-5日走勢
3. 脈衝量後續量縮不跌 = 多方控盤
4. 脈衝量後續量縮下跌 = 出貨完成

---

## 七、量滾量（換手量）

### 定義
在一段期間內，成交量持續維持在均量以上，但價格變動不大，像是在「滾動」籌碼。主力在特定價位區間反覆買賣，吸收浮額。

### 識別方法
1. 連續5日以上成交量 > 5日均量
2. 價格波動 < 5%
3. 常出現在底部或突破前的盤整區

### 主力量滾量的兩種目的

| 目的 | 特徵 | 後續走勢 |
|------|------|----------|
| 吸貨 | 量滾量期間低點逐步墊高 | 後續帶量突破上漲 |
| 出貨 | 量滾量期間高點逐步降低 | 後續跌破支撐下跌 |

### 判斷方法
1. 看量滾量期間的分點買賣超：主力買超 = 吸貨
2. 看融資變化：融資增加 = 散戶進場 = 主力出貨
3. 看K線型態：低點墊高 = 多頭換手

## 八、成交量型態的組合判讀

### 底部反轉的量能序列
1. 長期下跌 → 窒息量出現
2. 窒息量後 → 帶量長紅（脈衝量或遞增量）
3. 帶量長紅後 → 凹洞量或U型量（換手）
4. 換手完成 → 量滾量或遞增量（攻擊）

### 頭部反轉的量能序列
1. 上升趨勢 → 量能開始衰減
2. 價格創新高但量縮（量價背離）
3. 爆量長黑或脈衝量後下跌
4. 反彈量縮（凹洞量但右側不放量）

### 型態組合的勝率統計

| 組合 | 勝率 | 備註 |
|------|------|------|
| 底部窒息量 + 帶量突破 | 70-80% | 最可靠的底部訊號 |
| 凹洞量 + 帶量長紅 | 65-75% | 需配合籌碼面確認 |
| U型量 + 頸線突破 | 60-70% | 需配合型態學確認 |
| 量滾量 + 主力買超 | 70-80% | 最穩健的吸貨訊號 |
| 脈衝量 + 高檔長黑 | 75-85% | 強烈頭部訊號 |
