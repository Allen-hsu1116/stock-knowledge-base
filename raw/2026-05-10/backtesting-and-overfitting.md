## 回測是什麼

Backtesting（回測）是將交易策略套用於歷史資料，以驗證策略的準確性和可行性。回測允許交易者在不冒真金白銀的情況下測試策略。

### 回測的核心假設

- 歷史績效不代表未來表現，但歷史模式有參考價值
- 回測假設股票會以類似歷史模式移動
- 成功的回測結果顯示策略在歷史上有效，但不保證未來有效

### 回測常見指標

1. **Net Profit/Loss** — 淨損益
2. **Return** — 總報酬率
3. **Risk-Adjusted Return** — 風險調整後報酬
4. **Market Exposure** — 市場曝險程度
5. **Volatility** — 波動率
6. **Maximum Drawdown (MDD)** — 最大回撤
7. **Sharpe Ratio** — 風險調整後報酬的業界標準

### 回測偏誤

#### 1. 選擇偏差（Selection Bias）

挑選特定股票和時間區間來測試策略，使結果看起來很好。模型本質上有缺陷——雖然回測結果為正，但僅因為模型被設計來完美擬合這些資料。

**避免方法**：使用多個不同時間區間和無偏見的代表性股票樣本。

#### 2. 前瞻偏差（Look-Ahead Bias）

在回測模型中納入了當時無法取得的資訊。例如：假設年報在12/31就能取得，但實際上可能要等到2月才公布。使用未來資訊會讓回測結果虛高。

**避免方法**：嚴格確認每筆資料的可用時間點，只用當時已知的資訊。

#### 3. 生存者偏差（Survivorship Bias）

只用至今仍存在的公司做回測，忽略了已下市的公司。結果會讓策略績效看起來比實際更好。

**避免方法**：使用包含已下市公司的完整資料庫。

#### 4. 過擬合（Overfitting）

過度調整參數使策略完美匹配歷史資料，但對未來資料毫無預測力。參數越多、歷史績效越完美，過擬合風險越高。

**避免方法**：
- 樣本內訓練（In-Sample）/ 樣本外測試（Out-of-Sample）分離
- 參數精簡原則（Occam's Razor）
- Walk-Forward Analysis（滾動前進分析）
- 蒙地卡羅模擬壓力測試

## Sharpe Ratio（夏普比率）

Sharpe Ratio = (Rx - Rf) / StdDev Rx

- Rx = 投資組合期望報酬
- Rf = 無風險利率
- StdDev Rx = 報酬率標準差（波動率）

### 夏普比率評分標準

| 夏普比率 | 評價 |
|----------|------|
| < 1 | 不佳 |
| 1 - 1.99 | 適當/良好 |
| 2 - 2.99 | 很好 |
| > 3 | 優秀 |

### 關鍵理解

- 高報酬不代表好投資——可能是承擔過高風險換來的
- Sharpe Ratio 把報酬除以風險，衡量每一單位風險換來的超額報酬
- 基金A報酬20%、Sharpe 2.0 vs 基金B報酬30%、Sharpe 0.5 → A是更好的投資

### 幾何夏普比率 vs 修正夏普比率

- 幾何夏普比率使用複合報酬的幾何平均數，更保守
- 修正夏普比率使用算術平均數，較常用
- 比較時要同類相比（幾何比幾何、修正比修正）

## Maximum Drawdown（最大回撤）

MDD = (Trough Value - Peak Value) / Peak Value

最大回撤衡量投資從最高點跌到最低點的最大跌幅。

### MDD 作為風險指標

- MDD 越低，投資波動越小、風險越低
- MDD 可與市場基準指數比較（個股 MDD < 指數 MDD 表示個股表現優於市場）
- 例如：Tata Motors MDD -40% vs NIFTY MDD -50%，個股雖然絕對值高但相對市場更好

### 回撤最佳化投資組合（Drawdown Optimized Portfolio）

- 用 MDD 替代波動率建構投資組合
- 不僅追求低 MDD，也確保回撤後恢復新高的平均時間在兩年以內
- 投資人關心的是：虧損後多久能回本？

## 回測實戰步驟

1. **選定策略邏輯** — 先有明確的進出場規則
2. **收集無偏見的歷史資料** — 包含已下市公司，確認資料可用時間點
3. **樣本內訓練（60-70%）** — 調整參數
4. **樣本外測試（30-40%）** — 驗證策略
5. **計算績效指標** — Sharpe、MDD、CAGR、勝率、最大連虧
6. **壓力測試** — 蒙地卡羅模擬、不同市場環境
7. **參數敏感度分析** — 微調參數看績效是否劇烈變化
8. **Walk-Forward 驗證** — 滾動訓練+測試
