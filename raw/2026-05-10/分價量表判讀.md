## 什麼是分價量表

分價量表（Price Level Volume Table / 分價量表）是將一段時間內每個成交價位的成交量、成交筆數列出，以直觀方式呈現價量分佈的工具。

在台灣股市，分價量表可在：
- 券商看盤軟體（如永豐金、群益等）
- GoodInfo! 台股資訊網
- CMoney 看盤軟體

## 分價量表的核心組成

| 欄位 | 說明 |
|------|------|
| 價格 | 每個成交價位 |
| 成交量 | 該價位的總成交量（張/股） |
| 成交筆數 | 該價位的成交筆數 |
| 均量/筆 | 成交量 ÷ 成交筆數，判斷大單還是小單 |
| 佔比 | 該價位成交量佔總成交量的百分比 |

## 分價量表的實戰解讀

### 1. 支撐壓力辨識
- **大量區 = 支撐/壓力區**：分價量表中成交量最大的價位，代表該價位累積了最多籌碼
- 上方大量區 = 壓力帶（套牢區）
- 下方大量區 = 支撐帶（成本區）
- **關鍵觀察**：股價穿越大量區的方向決定是支撐還是壓力

### 2. 主力成本區判斷
- 主力進場價位通常伴隨大筆成交（均量/筆數大）
- 如果某價位出現「量很大、筆數很少、均量/筆很高」→ 大單進場
- 主力成本區通常出現在底部區間的量能密集帶

### 3. 出貨訊號
- 高價區出現大量、大筆成交 → 主力可能在高位派發
- 高價區出現大量、但均量/筆小 → 散戶追價、主力拆單出貨
- 對比：低價區大單 vs 高價區散單 → 典型的主力出貨型態

### 4. 突破確認
- 股價突破壓力帶時，看壓力帶的量是否被消化
- 突破後回測壓力帶（壓力轉支撐），若壓力帶量已被消化（回測量縮）→ 真突破
- 突破後回測壓力帶量未消化（回測帶量）→ 可能假突破

### 5. 區間換手判讀
- 盤整期間分價量表呈現均勻分佈 → 健康換手
- 盤整期間集中在單一價位 → 可能有主力控盤
- 集中在上方 → 壓力重；集中在下方 → 支撐強

## 分價量表與 Market Profile 的關聯

分價量表是 Market Profile（市場輪廓）的基礎數據：
- Market Profile 將分價量按時間分佈呈現（TPO圖）
- 分價量表是簡化版，只看量的分佈不看時間
- 兩者結合使用效果更佳：分價量表看「多少量」，Market Profile 看「何時成交」

## 分價量表 + 籌碼面交叉驗證

| 分價量表訊號 | 籌碼面驗證 | 結論 |
|-------------|-----------|------|
| 底部大單進場 | 外資/投信連續買超 | 主力進場確認 |
| 高價區散單追價 | 法人買超減少或翻賣 | 主力出貨警告 |
| 突破帶量消化壓力 | 千張大戶增加 | 真突破確認 |
| 突破量縮未消化 | 融資增加、大戶減少 | 假突破風險 |

## 注意事項

1. **分價量表有時間限制**：日分價量表只看當天，週分價量表覆蓋一週，選對時間框架很重要
2. **除權息調整**：除權息後的價格需要調整，否則會出現虛假的大量區
3. **流動性低的股票**：成交量太小時分價量表意義有限，容易被單一大單扭曲
4. **主力拆單**：主力可以將大單拆成多筆小單，讓均量/筆不明顯，需搭配分點追蹤
5. **不要單獨使用**：分價量表要搭配K線、均線、籌碼面一起判斷
