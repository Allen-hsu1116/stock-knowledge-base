## 均值回歸的核心原理

均值回歸理論認為資產價格，或者某些資產組合的價格，雖然短期會隨機波動，但長期來看，會傾向於回歸到一個歷史平均值或均衡水平。比喻：主人（長期均衡價值）、小狗（實時價格）、繩子（拉回均值的市場力量）。

### 時間序列的「定常性」

一個序列的統計特性（如均值、變異數）如果不會隨著時間改變，稱之為具有定常性 (Stationarity)。透過自回歸過程 AR(1)：

X(t) = φ * X(t-1) + ε(t)

- |φ| < 1：序列定常，具均值回歸特性
- |φ| = 1：隨機漫步，非定常
- |φ| > 1：發散

## 統計學驗證均值回歸可行性

### ADF 檢定 (Augmented Dickey-Fuller Test)

- 虛無假設 (H0)：時間序列存在單位根（隨機漫步，無均值回歸）
- 對立假設 (H1)：時間序列不存在單位根（定常，有均值回歸）
- p-value < 0.05：拒絕 H0，序列具均值回歸特性

大部分單一金融資產的價格序列都是非定常的，直接應用均值回歸策略風險極高。

### 共整合與配對交易 (Pairs Trading)

兩個或多個本身為非定常的時間序列，其某種線性組合（價差）卻是定常的，則存在共整合關係。常見配對：

- 同產業競爭對手：台積電 vs 聯電，可口可樂 vs 百事可樂
- 不同級別的公司債：AAA vs A
- 相似經濟驅動因素的貨幣對：AUD/JPY vs CAD/JPY

共整合 ADF 檢定（Engle-Granger 檢定）步驟：
1. 對兩個資產價格序列進行線性迴歸，得到殘差序列（價差）
2. 對殘差序列進行 ADF 檢定
3. 若殘差序列定常（p-value < 0.05），則存在共整合

## 實戰操作：Z-Score 與布林通道

Z-Score = (當前價差 – 價差的移動平均) / 價差的移動標準差

常見交易規則：
- Z-Score > 2.0（或 2.5）：價差過度偏離，做空價差
- Z-Score < -2.0（或 -2.5）：價差過度偏離，做多價差
- Z-Score 回歸 0 附近：平倉
