## 期貨未平倉量（Open Interest, OI）

### 什麼是未平倉量

未平倉量（Open Interest, OI）是期貨與選擇權市場中，已成交但尚未結算的合約總數量。一份未平倉合約包含一個買方和一個賣方，但只計算為1口（不是2口）。

OI 反映的是留在市場裡的資金規模，可以看作是投資人「押注」的總量，而非單日的交易熱度。

### OI 與成交量的區別

- **成交量（Volume）**：當日成交的合約數量，反映交易熱絡程度，隔日歸零重算
- **未平倉量（OI）**：市場上尚未結清的合約總量，持續累計直到合約到期結算歸零

三種交易情況對 OI 的影響：
1. 新買方+新賣方建倉 → OI +1
2. 原有持倉者之間換手 → OI 不變
3. 原有持倉者雙方平倉 → OI -1

### OI 與價格變動的關係（核心判讀）

| OI 變化 | 價格變化 | 市場情緒 | 趨勢解讀 |
|---------|---------|---------|---------|
| OI 上升 | 價格上漲 | 多頭趨勢強 | 新資金進場，買方信心強 |
| OI 上升 | 價格下跌 | 空頭趨勢強 | 資金押注下跌，空頭力道增 |
| OI 下降 | 價格上漲 | 趨勢可能轉弱 | 交易者平倉離場，趨勢尾聲 |
| OI 下降 | 價格下跌 | 趨勢可能轉弱 | 賣壓減少，可能進入震盪或反彈 |
| OI 持平 | 價格震盪 | 市場缺乏方向 | 多空勢力均衡，需等待新訊號 |

重點口訣：
- **OI 上升 = 新資金湧入，趨勢可能延續**
- **OI 下降 = 市場降溫，趨勢可能反轉**
- **量價齊揚（OI升+成交量升）= 趨勢強勁**
- **OI升+成交量降 = 交易熱情減弱，小心趨勢陷阱**

### 三大法人 OI 觀察重點

台灣期交所每日15:00公布三大法人（外資、投信、自營商）的期貨未平倉數據：

- **外資 OI**：通常引領大盤趨勢，外資 OI 淨多單持續增加 = 偏多訊號
- **投信 OI**：相對穩定，變化較小
- **自營商 OI**：偏避險操作
- **散戶 OI**：常被視為「反向指標」，散戶極度看多時反而要警惕見頂

**多空淨額判讀**：
- 多空淨額 > 0（正值）：市場偏多
- 多空淨額 < 0（負值）：市場偏空

### 散戶多空比計算

散戶的小型台指期貨未平倉多單數量 = 小型台指期未平倉總量 – 三大法人小台多方未平倉量
散戶的小型台指期貨未平倉空單數量 = 小型台指期未平倉總量 – 三大法人小台空方未平倉量

(散戶多單 – 散戶空單) / 小型台指期未平倉總量 = 台指期散戶多空比

行情下跌時，散戶多空比正值越大（散戶看多），行情可能反彈。
行情上漲時，散戶多空比負值越大（散戶看空），行情可能續漲。

### 高未平倉 vs 低未平倉的市場意義

**高未平倉量**：
- 流動性增加，買賣更容易成交
- 趨勢延續可能性較高
- 潛在波動性增加（關鍵技術位突破時可能引發連鎖反應）

**低未平倉量**：
- 市場趨勢可能轉弱
- 波動性降低，可能進入震盪區間
- 流動性降低，進出場更困難

### OI 基本特性

1. 沒有數量上限（不像股票有流通股數上限）
2. 通常隨時間緩慢增加，到期日歸零
3. 近月 OI > 遠月 OI（近月合約交易更活躍）

## 選擇權未平倉量判讀

### 選擇權 OI 以「賣方」角度解讀

選擇權未平倉量通常以「賣方」角度解讀，因為賣方需要負擔較高的保證金，且承擔低機率大行情損失的風險。

例如：15600 Call 的未平倉量有3萬口，重點放在 Sell Call 3萬口 → 偏向看不漲、看有壓力。

- **Call OI 增加** → 不看漲、看有壓力的態度增強
- **Put OI 增加** → 不看跌、看有支撐的態度增強

### 選擇權 Put/Call Ratio（PCR）

**計算公式**：
PCR = Put 全體市場未平倉量 / Call 全體市場未平倉量

**判讀方式有兩派**：

**第一派：直接解讀**
- PCR > 1：Put OI > Call OI，市場氣氛偏空
- PCR < 1：Call OI > Put OI，市場氣氛偏多

**第二派：反向指標（散戶指標）**
選擇權市場多為散戶參與，PCR 被解讀為散戶指標：
- PCR 數值越高 → 散戶偏空 → 反向看多
- PCR 數值越低 → 散戶偏多 → 反向看空

**成交量 PCR vs 未平倉量 PCR**：
- 成交量 PCR：Put 成交量 / Call 成交量，反映短期交易情緒
- 未平倉量 PCR：Put OI / Call OI，反映中長期留倉心態，更具參考價值

### 選擇權未平倉大量區（支撐壓力判斷）

1. 找出 Call 未平倉量最大的履約價 = 上檔壓力區
2. 找出 Put 未平倉量最大的履約價 = 下檔支撐區
3. 兩個區間形成一個支撐壓力帶

**進階判斷**：
- 參考鄰近履約價的 OI 分布狀況
- 觀察單日 OI 增減變化
- 大量區 + OI 明顯增加 = 支撐/壓力效果更顯著

### 三大法人選擇權未平倉觀察

期交所公布外資、投信、自營商的選擇權 Call/Put 未平倉部位，觀察法人留倉狀況可衡量其對後市的樂觀/悲觀程度。
