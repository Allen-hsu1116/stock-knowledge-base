## 原文內容整理

### 三大報表概覽

**資產負債表（Balance Sheet）** — 靜態快照
- 某一特定時間點的財務狀況
- 公式：資產 = 負債 + 股東權益
- 資產：公司擁有的一切（現金、應收帳款、存貨、設備等）
- 負債：公司必須償還的義務（流動負債、長期負債）
- 股東權益：資產減去負債後的淨值（股本、資本公積、保留盈餘）

**損益表（Income Statement）** — 動態成績單
- 一段時間內的盈利與虧損
- 公式：稅後淨利 = 營收 - 成本 - 費用 - 稅
- 營業收入 → 營業成本 → 營業毛利 → 營業費用 → 營業利益 → 業外收支 → 稅前淨利 → 稅後淨利

**現金流量表（Cash Flow Statement）** — 現金真相
- 一段時間內現金流入和流出
- 三大現金流：營業現金流、投資現金流、融資現金流
- 營業現金流 = 日常營運產生的現金
- 投資現金流 = 購買/出售長期資產
- 融資現金流 = 借款/還款/發放股利

### 三表勾稽關係

1. **損益表的淨利 → 影響資產負債表的「保留盈餘」**
   - 損益表稅後淨利會計入保留盈餘
   - 保留盈餘是股東權益的一部分

2. **資產負債表的變動 → 影響現金流量表的各項目**
   - 現金及約當現金的增減 = 現金流量表的期末餘額
   - 應收帳款增加 → 營業現金流減少（錢還沒收進來）
   - 存貨增加 → 營業現金流減少（錢卡在庫存）
   - 固定資產增加 → 投資現金流減少（買了設備）

3. **現金流量表 → 解釋資產負債表中「現金」的增減**
   - 期初現金 + 淨現金流 = 期末現金
   - 現金流量表把資產負債表期初到期末的現金變化說清楚

### 交叉驗證重點

**假性獲利警訊**：
- 損益表有淨利，但營業現金流為負 → 獲利被應收帳款或存貨卡住
- 淨利成長但營業現金流持平或下降 → 獲利品質差
- 應收帳款增幅遠超過營收增幅 → 可能虛增營收或收不到錢

**償債能力判斷**：
- 流動比率（流動資產/流動負債）> 1.5 為佳
- 負債比率（負債/資產）< 50% 較安全
- 但要搭配現金流量表確認營業現金流是否足以還債

**獲利品質驗證**：
- 毛利率穩定或上升 = 產品競爭力強
- 營業利益率 = 本業獲利能力（排除業外）
- 營業現金流/稅後淨利 > 1 = 獲利品質好（現金真的收進來了）

**資本結構分析**：
- 看資產負債表的負債/權益比
- 看現金流量表的融資現金流：是靠借錢還是靠營運？
- 長期投資現金流為負可能是擴張，但要看營業現金流是否支撐

### 三表一起看的閱讀順序

1. **先看損益表**：公司賺不賺錢？毛利率、營利率、淨利率趨勢
2. **再看資產負債表**：公司穩不穩？負債比率、流動比率、應收帳款/存貨變化
3. **最後看現金流量表**：錢真的進來了嗎？營業現金流是否為正？是否足以支撐投資和還債？
4. **交叉驗證**：三表數字是否一致？有沒有矛盾？

### 常見矛盾與警示

| 矛盾現象 | 可能問題 |
|---------|---------|
| 損益表賺錢但現金流為負 | 應收帳款或存貨堆積 |
| 營收成長但毛利率下降 | 成本失控或低價搶市 |
| 淨利成長但營業利益下降 | 靠業外收入支撐 |
| 保留盈餘增加但現金減少 | 盈餘品質差或大量投資 |
| 資產增加但負債增加更快 | 槓桿過度 |
| 營業現金流持續為負 | 即便損益表好看也終將倒閉 |

### 快速財報健檢清單

- ✅ 流動比率 > 1.5
- ✅ 營業活動現金流為正
- ✅ 毛利率穩定或上升
- ✅ 應收帳款週轉天數合理（< 60天）
- ✅ 負債比率 < 50%
- ✅ 營業現金流 > 稅後淨利（獲利品質佳）
