## 波浪理論是什麼？

波浪理論是由美國會計師兼作家拉爾夫·納爾遜·艾略特(Ralph Nelson Elliott)於1938年提出。

艾略特受到道氏理論啟發，認為市場的走勢就像潮汐波浪一樣，一波接著一波，他觀察波浪模式，找到一些重覆發生的結構特性。

波浪理論主要用來判斷趨勢進行的階段，以及趨勢是否反轉。但因為很難做精確分析，所以常搭配其他技術分析指標運用。

## 波浪理論的適用範圍

- 適用分析指數、外匯、期貨等大規模市場
- 容易被人為操控、股本小、個股不適用
- 主要用於中長期的走勢分析
- 以自然法則為主，不涉及人為干涉

## 艾略特波浪理論的規則

完整一個波浪總共有8波，基本架構是由5個推動波(Impulse Waves)、3個調整波(Corrective Waves)組成。

### 每個波浪的特性

| 波浪 | 特性 |
|------|------|
| 波浪1 | 第一波在開始時通常不明顯，這時基本面消息幾乎是負面消息，市場情緒調查明顯看跌 |
| 波浪2 | 通常是波浪1的回檔走勢，但永遠不會超過波浪1的起點。成交量會比波浪1小，價格通常不會超過波浪1的61.8% |
| 波浪3 | 這是道氏理論中的買入信號，趨勢進入牛市，通常會產生最大的成交量、價格變動，也是5個推動波中最長的 |
| 波浪4 | 與波浪2很像，是平緩的盤整或回檔，底部不會低於第1浪的高點。回檔幅度通常低於波浪3 |
| 波浪5 | 可能比波浪3的波動小，也可能比波浪3大。其他技術指標逐漸出反向訊號。市場情緒高漲，許多普通投資者會買入在高點 |
| 波浪A | 投資人普遍認為下跌只是短暫回檔，是牛市的修正 |
| 波浪B | 呈現價格向上的假反彈，許多人會進場搶低點 |
| 波浪C | 市場情緒開始意識到這是熊市，波浪C通常和波浪A一樣大，有一種說法是可以延伸到A浪的1.618倍 |

## 多頭行情基本規則

- 波浪2的最低點不低於波浪1起點
- 波浪3通常不會是最短的波段，且大多是主升段
- 波浪1最高點與波浪4最低點不能重疊
- 波浪b反彈上升高點，不會超過波浪a原本最高點

## 延伸波

當符合波浪理論的基本規則後，如果價量波動變大很多，就會產生出延伸波，而且只會出現1次。

延伸波主要特性：
- 出現在1、3、5波浪裡，其中1波，且只會出現1次
- 當有延伸波出現時，表示走勢非常強勁且趨勢較長
- 符合股價大漲、量能放大的條件，通常出現在第3波的機率高

## 邪惡的第5波

波浪5也可以存在很長的延伸波，這時通常市場已經大漲很長一段時間，氣氛很高漲，許多新手紛紛進場，第5波可以漲到天上去，幅度甚至超越第3波。

但隨後會迎來強烈的反轉，把之前漲的都跌掉，前面漲多兇後面ABC波就殺多兇。這段第5波的漲幅，也被稱為「邪惡的第5波」，意思是如果成功參與可能有很大的利潤，但不一定能即時脫身。

## 波浪理論的波浪計算方式

| 波浪 | 計算方式 |
|------|----------|
| 波浪1 | 第1個推動浪，主要用來計算其他不同波浪長度的標準 |
| 波浪2 | 範圍通常在波浪1的38.2%和78.6%之間 |
| 波浪3 | 通常是最長的波浪，可能是1.618×波浪1、2.62×波浪1、4.25×波浪1 |
| 波浪4 | 波浪3的23.6%或38.2%或50%，23.6%、38.2%較常見 |
| 波浪5 | 如果波浪3≥1.618，波浪5=波浪1或1.618×波浪1或2.62×波浪1；如果波浪3<1.618，第5浪往往會延伸 |
| 波浪A | 第1個修正浪，用來計算其他不同波浪長度的標準 |
| 波浪B | 通常回檔A浪的38.2%到78.6% |
| 波浪C | 波浪A長度的1.382或1.618倍 |

## 如何將波浪理論運用在交易中

- 波浪理論的原則主要依賴市場情緒，可搭配恐懼和貪婪指數
- 最佳進場時機是猜測第3波剛形成，第2波結束後第3波即將突破時
- 另一個時機是在第4波結束、第5波即將開始突破第3波前高時
- 波浪事後看有那麼回事，但事先預測其實很難做到精準

## 使用波浪理論的注意事項

- 波浪模式並不一定會在市場上發生，只是提供一個可能情況
- 建議與其他技術指標和工具相結合
- 這是1930年代提出的理論，市場心理學、人群行為或集體思維或許隨時代演變已出現變化

市場先生觀點：波浪理論提供一種思考方式，思考趨勢與人群心理一連串變化的結合，從希望到興奮、從興奮到狂熱、從狂熱再到懷疑、從懷疑到恐慌、再從恐慌到絕望。

強烈建議：不要試圖精準預測價格會走到哪裡。不要太把費波那契數列這種神祕學的東西當成原則。
