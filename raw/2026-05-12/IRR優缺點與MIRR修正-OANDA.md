## 什麼是內部報酬率IRR?

內部報酬率（英文：internal rate of return，簡稱IRR），也稱年化報酬率。是一種衡量投資收益率的評估方法，計算的原理是資產潛在的價值(Future value)以多少的折現率來計算，可以得到資產的現值(Present value)。

折現率 (discount rate) 指的是在將未來的現金流折算成現值的比率，折現率越高，代表未來預期現金流量的現值越低，所以折現率與價值成反比。

簡單來說內部報酬率 (IRR) 就是在投資的每個期間內所投資的每一塊錢所賺取的百分比。

## 內部報酬率的計算公式

IRR 公式是將淨現值設置為零的利率下，計算出投資貼現在未來的現金流。

0 = NPV = Σ CFₙ / (1 + IRR)ⁿ

N = 投資持有期間，n = 0 到 N 之間的每個時期，CFₙ = 所投資的現金流，IRR = 內部報酬率。

## 單筆投資與定期定額投資用IRR計算的差異

同樣一筆資金不同方式投入也會有不同的IRR報酬率。

年初投資$100,000元，過了10年後本利和$200,000元，與每年投資$10,000元，過了10年後本利和$200,000元。

單筆投資過了10年IRR報酬率為7.1773%，而定期定額10年後IRR報酬率為12.3041%。

定期定額的IRR較高，因為每期投入金額的時間價值不同——後期投入的錢「在市場裡的時間較短」，但同樣貢獻了最終的報酬。

## 內部報酬率的優點

1. 透過 IRR 評估投資計畫可行性時，能看出該投資計畫所能帶來的報酬率，決策較為簡單明確。
2. IRR 可以用在相同基礎上對不同投資計畫的報酬進行評估，可以依序排序。

## 內部報酬率的缺點

1. IRR 計算上，若現金流多次正負變動，可能出現多個IRR的情況，導致在適用上有所限制。
2. IRR 是「內部」指標，不考慮外部因素：如通貨膨脹或潛在成本變動。
3. 比較不同投資計畫的時間長短時，短期計畫的IRR通常較高，長期計畫的IRR通常較低，這不一定是投資潛在報酬的實際情況。

## 如何使用 IRR？結合淨現值（NPV）判斷投資可行性

### 第一步：計算 IRR
根據投資案的預估現金流，計算出使 NPV 為零的折現率，即為該案的 IRR。

### 第二步：設定門檻報酬率（Hurdle Rate）
投資人應設定一個參考標準（例如公司的 WACC 或自身期望報酬率），作為判斷依據。若 IRR > 資金成本，屬於「具吸引力」的投資案。

### 第三步：搭配 NPV 進行決策

| 條件 | 解讀 |
|------|------|
| IRR > 資金成本，NPV > 0 | 投資具吸引力，建議進行 |
| IRR > 資金成本，但 NPV ≈ 0 | 邊際投資，需觀察其他風險因素 |
| IRR < 資金成本，NPV < 0 | 投資不符預期，應避免 |

## 什麼是修正內部報酬率（MIRR）？為何它比 IRR 更實用？

MIRR（Modified Internal Rate of Return）是一種更貼近現實的評估方式。

### MIRR 的特點

1. MIRR 假設正向現金流的再投資報酬率為投資者可控制的「再投資率」
2. MIRR 對負向現金流採用資金成本折現
3. MIRR 永遠只有一個解，不會出現多重解的問題

### MIRR vs IRR 範例

某投資案初期投入 -10,000，未來三年分別回收 4,000、5,000、3,000：

- 再投資報酬率 8%，資金成本 6%
- IRR 計算結果約 13.1%
- MIRR 計算結果約 10.8%

MIRR 所呈現的報酬率更保守也更實際，避免高估投資吸引力。

### MIRR 使用情境

- 現金流不規則、含多次正負變化 → IRR 可能出現多重解，MIRR 始終唯一
- 投資者無法保證再投資可達 IRR 水準 → MIRR 可指定實際再投資利率
- 企業大型資本預算或基礎建設專案分析 → MIRR 反映資金成本與再投資成本
