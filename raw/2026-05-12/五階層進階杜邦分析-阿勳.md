## ROE 的意義

ROE = 稅後淨利 / 平均股東權益

每股盈餘(EPS)衡量賺多少錢，ROE衡量多會賺錢。

舉例：
- 肉燥公司：本金1000萬，賺500萬，ROE = 50%
- 咖啡公司：本金5000萬，賺800萬，ROE = 16%

ROE 比 EPS 更具參考性。

## ROE 三大缺陷

1. **無法看出舉債程度**：自有資金1000萬+舉債5000萬+賺500萬，ROE看似50%，但風險極高
2. **無法看出資產使用效率**：不知道翻桌率
3. **無法看出業外損益**：本業300萬+業外當沖200萬，合計500萬，但業外不可持續

## 傳統杜邦分析（三因子）

ROE = 淨利率 × 資產週轉率 × 財務槓桿

- **淨利率** = 稅後淨利 / 營業收入（收100塊剩多少）
- **資產週轉率** = 營業收入 / 總資產（每1元資產帶來多少營收）
- **財務槓桿** = 總資產 / 股東權益（舉債程度）

判斷原則：
- 高ROE + 高淨利率 → 好事
- 高ROE + 高資產週轉率 → 好事
- 高ROE + 高槓桿 → 不一定是壞事，看產業平均

但傳統杜邦仍無法區分本業與業外獲利。

## 阿勳五階層進階杜邦分析

ROE = 稅賦優勢 × 業外貢獻比 × 本業利益率 × 資產週轉率 × 財務槓桿

### 五因子解釋

1. **稅賦優勢** = 稅後淨利 / 稅前淨利
   - 就是「1 - 稅率」
   - 越高越好，表示被課的稅越少
   - 最大值為1（不課稅），不會大於1

2. **業外貢獻比** = 稅前淨利 / 營業利益
   - =1：沒有業外損益（最理想）
   - >1：業外獲利
   - <1：業外損失
   - 越高代表業外獲利越多，但業外不可持續

3. **本業利益率** = 營業利益 / 營業收入
   - 不含稅率和業外的純正獲利能力
   - 越高越好，代表本業越賺錢

4. **資產週轉率** = 營業收入 / 總資產
   - 資產使用效率
   - 薄利多銷型公司週轉率高

5. **財務槓桿** = 總資產 / 股東權益
   - 舉債程度
   - 要跟同業比較

### 判斷原則

- 高ROE + 高稅賦優勢 → 好事
- 高ROE + 高資產週轉率 → 好事
- 高ROE + 高槓桿 → 不一定，要與同業比較
- 高ROE + 高業外獲利比 → **壞事**
- 高ROE + 高本業利益率 → 好事

### 商業模式辨識

- 賺取品牌、技術財 → 利潤率很高
- 賺取管理財 → 利益率低，但資產週轉率高
- 控股型公司 → 業外獲利比高
- 財務操作厲害的公司 → 稅賦優勢比與槓桿比率都很高

### 實戰案例

**台積電(2330)**：好的商業模式提升
- ROE提升主要來自利益率上升與槓桿提升
- 半導體製程推進→利潤率越來越高
- 舉債擴充產能應付全球供不應求

**樺漢(6414)**：不好的商業模式提升
- ROE提升來自業外獲利比上升與稅賦優勢提升
- 本業利潤率卻在下滑
- 2022年Kontron IT Service停業一次性認列處分利益121.6億
- 業外獲利不可持續，股價沒有相應反應

**同業比較案例**

隱形眼鏡雙雄：晶碩(6491) vs 精華(1565)
- 晶碩在本業利潤率、業外獲利比、稅賦優勢、資產使用效率都優於精華
- 晶碩負債僅3成，精華過度保守

連鎖藥局雙雄：大樹(6469) vs 杏一(4175)
- 大樹在本業利潤率、業外獲利比、資產使用效率與槓桿程度都優於杏一
- 這是大樹股價爆衝的主因

### 一次看5年趨勢

ROE是流量指標，單一年度容易有偏誤，應該一次看5年趨勢，了解企業賺錢效率的變化來自哪裡。
