# XQ跨頻率多重濾網交易策略設計

**來源**：https://www.xq.com.tw/lesson/sensor/multifilters/
**日期**：2026-05-11
**主題**：技術分析

---

Alexander Elder 的三重濾網交易系統概念：

1. 掌握主要趨勢（週線）
2. 在短期超買超賣後尋找進場時機（日線）
3. 從股價波動中確認進場點（盤中）

這等於同時使用週線、日線及盤中洗價（例如60分鐘線），是一個跨三種頻率的交易策略。

### 在XQ上建構三重濾網

**選股腳本1 - 週線移動平均線多頭排列**：
```
input:S1(6,"天期");
value1=average(close,s1);
if close>value1
then ret=1;
```
應用在週線上。

**選股腳本2 - 日線KD進入超賣區**：
```
input: Length(9), RSVt(3), Kt(3);
variable: rsv(0), k(0), _d(0);
Stochastic(Length, RSVt, Kt, rsv, k, _d);
IF k<20 and _d<30
then ret=1;
```
應用在日線上。

**策略雷達3 - 60分鐘線創新高**：
```
input: Price(close);
input: Length(10);
if Price > highest(high[1],Length) then ret=1;
```
應用在60分鐘線。

### 執行流程

1. 用前兩個腳本選出符合條件的股票，命名為「三重濾網之前兩重」
2. 讓系統每日自動執行選股
3. 將第三個腳本納入策略雷達，盤中即時警示

### 注意事項

- 市場老人常說：不要逆勢操作、用長線保護短線、看長作短
- 三重濾網系統的設計概念與市場老手想法不謀而合
- 第一重建議用趨勢型指標（MA、MACD）
- 第二重建議用震盪指標（KD、RSI）
- 第三重建議用突破型交易策略
- 目前XS系統限制：前兩重必須日線以上，無法執行日線+60分+1分K的組合
