# 成交量分佈分析 Volume Profile 進階實戰

成交量分佈圖（Volume Profile，簡稱 VP）是一種將累積成交量以橫條形式顯示在特定價格區域上的技術指標。它能夠清晰呈現價格在某一區間內的成交量，是交易者在理解市場結構、投資者情緒和股票成本的重要工具之一。在技術分析中，VP 雖然無法直接預測趨勢，但可以有效幫助交易者了解市場中的重要價位和參與者的意圖，是一個很好的輔助和確認性工具。

## Volume Profile 的基本組成

- 主要成交價 (POC)：這是成交量最高的價格，通常顯示為橫條最長的位置，也會有紅線標註。POC 代表交易者普遍認同的價格，市場對此價格的支持度最大。

- 主要交易區間 (Value Area, VA)：指以 POC 為中心，累積成交量達到 70% 的價格範圍。VA 代表了市場上多數參與者認同的價位區域，對於市場價格走勢有較大的影響力。

- 主要交易區高點 (VAH) 和 主要交易區低點 (VAL)：分別是主要交易區間的最高價和最低價。

- 低成交量區域 (Low Volume Node, LVN)：成交量低於平均的價位，這些區域通常是價格快速突破或回撤的結果，被視為市場中的"不認同價格區域"。

## Volume Profile 介紹

Volume Profile 可分為以下幾種顯示方式：

- 固定時間範圍 (Fix Range)：可以框選特定的時間範圍來計算 VP，適合於回顧特定時段的交易情況。

- 目前可視範圍 (Visible Range)：VP 根據交易者在圖表上選取的可視範圍來計算，這是最常使用的選項，能夠即時反映當前市場狀態。

- 每個交易日 (Session Volume)：顯示每個交易日的成交量分佈，有助於了解日內重要的價位。

### Volume Profile 的三種市場定律

Volume Profile 通過成交量的分佈揭示了市場中的三種定律：

- 回歸均值：價格大多數時候會回到市場認同的"公認價格區域"，在這個過程中，價格會持續在 VA 內震盪。當價格偏離主要成交價（POC）過遠時，通常會回歸到該區域，形成均值回歸現象。

- 回檔支撐：當股價處於上升趨勢中，價格回到 VA 或 POC 位置時，會出現支撐，因為此價位被視為合理的買入點。這些位置通常會吸引大量買盤，形成強有力的支撐。

- 反彈阻力：在下跌趨勢中，當價格反彈至 VA 或 POC 位置時，投資者通常會選擇賣出以減少損失，形成阻力。這是因為很多投資者的成本位於這些位置，他們希望在價格回到原成本時平倉以避免更大的損失。

## Volume Profile 的應用場景

### 1. 區間突破進場

當價格在低成交量區（LVN）附近進行突破時，通常會伴隨著大幅波動。這是因為在低成交量區域，市場上的交易認同度較低，價格很容易快速移動。交易者可以利用這些突破信號進行順勢交易，在確認突破成功後進場，捕捉後續的價格動能。

### 2. 區間逆勢操作

在區間震盪的市場中，交易者可以利用主要交易區間的高點（VAH）和低點（VAL）進行逆勢操作。在價格接近 VAH 時做空，接近 VAL 時做多，從中捕捉反彈或回撤的機會。這種操作適合於盤整市，因為在這樣的市場中，價格往往難以形成持續的趨勢，更多是在一個區間內波動。

### 3. 回測支撐與阻力

當股價突破某個重要價位（如 POC 或 VAH）後，經常會回測該價位以確認支撐或阻力的有效性。第一次的回測往往是強力的進場點，尤其是在回測之後，股價能夠再次突破上行，這意味著該價位獲得了市場的認同。

## 如何將 Volume Profile 與其他技術指標結合

Volume Profile 本質上是一個確認性的指標，因此在使用時通常需要與其他技術指標結合，例如趨勢分析（移動均線、趨勢線）或型態分析（頭肩頂、杯帶柄等）。先用這些工具判斷市場的趨勢，再利用 Volume Profile 來確認市場中最重要的成交價位，這樣可以提高交易決策的可靠性。

例如，在使用 ATR 突破策略時，可以結合 VP 來確認價格是否在 POC 之上並且沒有重大阻力區，這樣可以提高突破成功的機會。同時，Volume Profile 也可以用來確認底部型態的有效性，幫助找到更精確的進場點。

## 總結

Volume Profile 是一個功能強大且直觀的工具，它可以幫助交易者更好地了解市場中的主要價格區域以及參與者的行為模式。在交易中，Volume Profile 並非用來直接預測價格，而是幫助交易者理解市場的運行結構，尤其是在重要價位上的成交情況。通過結合趨勢分析和型態分析，Volume Profile 可以幫助交易者確認支撐與阻力位置，進而提高交易成功的概率。無論是順勢交易還是逆勢操作，了解 Volume Profile 都能幫助交易者在不同的市場環境中找到更優化的進場和出場時機。
